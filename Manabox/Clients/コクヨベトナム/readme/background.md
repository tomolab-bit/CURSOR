# プロジェクト背景・経緯

## プロジェクトの背景

### コクヨベトナムの事業概要
コクヨベトナム（KOKUYO VIETNAM）は、文房具やオフィス家具の製造・販売を行う企業として、ベトナム国内で安定した事業を展開しています。ハイフォン市に本社と工場を構え、ハノイ市とホーチミン市に代表事務所を設置し、3拠点での事業運営を行っています。

### ベトナム税務環境の変化
ベトナムでは近年、税務調査の動向が変化し、税務リスクマネジメントの重要性が高まっています。特に以下の要因により、税務コンプライアンスの強化が急務となっています：

- 税務調査の頻度増加と厳格化
- 移転価格税制に関する規制の強化
- 税制改正の頻繁な実施
- 税務申告の誤りによる罰金事例の増加

## プロジェクト開始の経緯

### 初期の課題認識
コクヨベトナムは、以下の課題を抱えていました：

1. **税務リスクの可視化不足**: 税務リスクの事前把握が困難
2. **税務レビューの非体系化**: 定期的な税務レビューの体制が未整備
3. **税制改正への対応遅れ**: 頻繁な税制改正への迅速な対応が困難
4. **税務データの分散管理**: 複数拠点の税務データの統合管理が未実現

### Manaboxとの関係性構築
コクヨベトナムは、これらの課題解決のため、ベトナムでの税務サポート実績が豊富なManaboxとの連携を決定しました。Manaboxの以下の強みが評価されました：

- ベトナム税務に関する豊富な知識と経験
- 四半期税務レビューの実績
- 税務リスクマネジメントの専門性
- 税務データ分析・可視化の技術力

## プロジェクトの目的・目標

### 主要目的
1. **税務リスクの最小化**: 税務リスクを事前に予測し、適切な対策を講じる
2. **税務コンプライアンスの強化**: 税務申告の精度向上とコンプライアンス確保
3. **税務業務の効率化**: 税務処理の自動化・効率化によるコスト削減
4. **経営判断支援**: 税務データ分析による経営判断の支援

### 具体的目標
- 四半期ごとの税務レビューの定期実施
- 税務リスク予測精度の向上（目標：85%以上）
- 税務申告精度の向上（目標：99%以上）
- 税務処理時間の短縮（目標：30%削減）

## プロジェクトの戦略的意義

### コクヨベトナムにとっての価値
- **事業安定性の確保**: 税務リスクの最小化による事業継続性の向上
- **競合優位性の確立**: 税務コンプライアンスの強化による信頼性向上
- **経営効率の向上**: 税務業務の効率化によるリソース最適化
- **将来成長の基盤**: 税務リスクマネジメント体制の確立

### Manaboxにとっての価値
- **製造業での実績構築**: ベトナム製造業での税務リスクマネジメント実績
- **技術力の実証**: AI活用税務リスク予測システムの実装実績
- **長期関係の構築**: 継続的な税務サポート関係の確立
- **他業界展開の基盤**: 他製造業への展開可能性の創出

## プロジェクトの進展状況

### Phase 1: 基盤構築期（2024年）
- 四半期税務レビュープロセスの確立
- 税務リスク評価フレームワークの構築
- 税務コンプライアンスチェックリストの作成

### Phase 2: データ活用期（2024年-2025年）
- 税務データ分析システムの導入
- 税務リスク予測モデルの構築
- 税制改正情報の自動収集システム

### Phase 3: 最適化期（2025年-現在）
- AI活用税務リスク予測システムの検討
- 税務データの可視化ダッシュボードの構築
- 税務トレーニングプログラムの実施

## 今後の展望

### 短期目標（1-3ヶ月）
- AI活用税務リスク予測システムの本格導入
- 税務データ統合プラットフォームの構築
- 税務ダッシュボードの機能拡充

### 中期目標（3-6ヶ月）
- 税務リスク予測精度のさらなる向上
- 税務トレーニングプログラムの実施
- 税務コンプライアンス監視システムの強化

### 長期目標（6ヶ月以降）
- 税務リスクマネジメントの完全自動化
- 他拠点への税務リスクマネジメント展開
- 税務データ分析による経営支援の拡充

このプロジェクトは、コクヨベトナムの税務リスクマネジメント体制の確立と、Manaboxの税務サポート技術力の実証を両立する、戦略的に重要な取り組みとなっています。
