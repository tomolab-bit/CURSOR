# 付録I：会計伝票書式体系（Phụ lục I: Hệ thống mẫu chứng từ kế toán）

## 通達99/2025/TT-BTC 付録Iの概要

本付録は、ベトナム会計通達99/2025/TT-BTCの**付録I（会計伝票書式体系）**に関する日本語版資料です。

### 会計伝票の種類

企業が会計処理を行う際に使用する主要な伝票は以下の通りです。

1. **Phiếu thu（収入伝票）**：現金・預金の収入を記録する伝票
2. **Phiếu chi（支出伝票）**：現金・預金の支出を記録する伝票
3. **Phiếu kế toán（振替伝票）**：勘定間の振替を記録する伝票
4. **Phiếu nhập kho（入庫伝票）**：在庫の入庫を記録する伝票
5. **Phiếu xuất kho（出庫伝票）**：在庫の出庫を記録する伝票
6. **Giấy báo Nợ（借項通知書）**：銀行からの借項通知
7. **Giấy báo Có（貸項通知書）**：銀行からの貸項通知
8. **Bảng kê（明細表）**：各種明細表
9. **Phiếu tính lương（給与計算書）**：給与計算に関する伝票
10. **Bảng tính và phân bổ khấu hao TSCĐ（減価償却計算・配賦表）**：固定資産の減価償却を計算する表

### 伝票の必須記載項目

各伝票には以下の必須記載項目が含まれます：

- **日付（Ngày）**：取引が発生した日付
- **伝票番号（Số phiếu）**：伝票の連番
- **相手先名（Tên đối tượng）**：取引相手の名称
- **金額（Số tiền）**：取引金額
- **内容（Nội dung）**：取引の内容説明
- **署名（Chữ ký）**：作成者・承認者の署名

### 伝票の管理

- 各取引ごとに1回のみ伝票を作成
- 伝票は時系列で管理し、欠番がないようにする
- 訂正が必要な場合は、元の伝票を残したまま訂正伝票を作成
- 伝票は法律で定められた期間保管する義務がある
