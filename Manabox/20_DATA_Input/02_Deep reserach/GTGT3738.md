1. 項目[37]・[38]に関する関連法令と根拠

。つまり納税者は、税務調査の開始決定が通知される前で、かつ決定に含まれない期間・内容についてのみ、10年以内であれば補足申告が可能です
luatvietnam.vn
luatvietnam.vn
。

2. 項目[37]・[38]記載の基本ルール

**項目[37]「過去期の控除VATの減額調整」および項目[38]「過去期の控除VATの増額調整」**は、いずれも申告書01/GTGTの第IV欄「前期からの控除VAT繰越額の調整」に位置するフィールドです
thuvienphapluat.vn
。基本的なルールは次のとおりです。

控除VAT繰越額の増減のみ対象: 項目[37]・[38]は仕入控除VAT（入力VAT）に関する増減調整専用です
thuvienphapluat.vn
。売上VAT（出力VAT）の過年度訂正には使用せず、売上に係る誤り（申告漏れや過大申告）は別途元の期間に対する補充申告等で対応します。一方、過去の誤りや調整が「控除可能VAT額」のみを増減させる場合に、この二つの項目が活用されます。

増額は[38]、減額は[37]: 前期以前からの繰越控除VATを**増やす場合は項目[38]に、その繰越額を減らす場合は項目[37]**に、それぞれ調整額を記入します
thuvienphapluat.vn
。増やすケースとは、過去に申告漏れだった仕入VATを今になって追加入力する場合など、過去期で本来もっと多く控除できたはずの額を取り戻すケースです。反対に減らすケースとは、過去に控除計上した仕入VATに誤り（控除対象外の請求書を含めていた等）があり、**控除額を取り消す（繰越額を減額する）**場合です。

現行期での申告調整: 前述の法令に基づき、原則として誤りが発生した当時の期ではなく、補正手続きを行う“現在の申告期間”において調整申告することが求められます
thuvienphapluat.vn
。例えば、2024年第1四半期の誤りを2024年第3四半期に発見した場合、補足申告（01/KHBS）は誤りのあったQ1/2024について作成しますが、そこで増減した控除VAT繰越額はQ3/2024（発見した期）の01/GTGTの[37]または[38]に計上します
thuvienphapluat.vn
。過去の申告データ自体は訂正されますが、その結果生じた繰越額の差分は現行期で調整するのがルールです。

調整額の根拠明示: [37]や[38]に金額を記入する場合、必ず対応する補足申告書（01/KHBS）や税務当局の決定書等で計算された増減額である必要があります
thuvienphapluat.vn
。これらの項目は独立して任意に入力するものではなく、裏付けとなる書類（補足申告の計算書、税務調査結果通知など）が存在します。税務ソフトHTKKでも、01/KHBSを作成するとその差額が自動的に当期の[37]/[38]欄に反映される仕組みになっています。

期限内の現行申告期で計上: 政令126号では「補足申告は当該期の申告期限内であれば現行期で調整可能」とも読める規定があります
gonnapass.com
（期限後は別途扱いの可能性）。ただし実務上、誤りを発見した時点でまだ当期申告期限が到来していない場合でも、その当期の申告書に[37]/[38]を用いて調整するのが一般的です
gonnapass.com
。一方、すでに申告期限が過ぎた期間について調整が必要な場合は、直近の未申告期（または現在の期）に計上することになります
gonnapass.com
。

以上の基本ルールに従えば、[37]・[38]欄への記載は**「過去の控除VAT繰越額の増減調整」を現行の申告期に反映する**明確な役割を果たします。

3. 実務でよくあるケースと項目[37]・[38]の使い方

次に、現場の会計・税務実務で頻出するケース別に、項目[37]・[38]の適切な扱いを整理します。

ケース① 補足申告による控除VAT繰越額の増減: 最も典型的なのは、過去の申告誤りを訂正する補足申告（01/KHBS）を行った結果、前期からの繰越控除VAT額に増減が生じたケースです。例えば過去に仕入税額の計上漏れがあり控除額を追加できる場合や、逆に誤って控除していた仕入税額を取り下げる場合です。いずれも補足申告の結果として前期繰越欄（01/KHBS様式IIの「未控除VAT調整額」）に正負の差異が発生します
thuvienphapluat.vn
。この差額がプラスであれば当期申告書の項目[38]に記入して控除繰越額を増やし、マイナスであれば項目[37]に記入して繰越額を減らします
thuvienphapluat.vn
。
根拠例: ハイズオン省税務局の公文書1592/CT-NVT（2025年）では「補足申告によって当該期の未控除VATに増減のみ生じる場合、補足申告書を作成のうえ、その増加額を項目[38]、減少額を項目[37]に現在の申告期で申告する」と明示されています
thuvienphapluat.vn
。これにより追加納税や還付返還を要さない単なる控除額の調整は、現行期での繰越調整で処理できることになります。

ケース② 税務調査や更正による控除VATの修正: 税務当局（税務局や監査機関）の調査・検査によって、過年度の控除VAT額の誤りが指摘され増減が発生するケースもあります。例えば、ある年の税務調査により「○年○月期の申告に含まれていた不適格な仕入VAT 100百万ドンを控除不許可とする」等の処分決定が出た場合です。この場合、納税者は自ら補足申告を提出する必要はありません（税務当局が集中管理システム上で調整）
gonnapass.com
が、決定によって減少した控除VAT額を当期申告書の項目[37]で控除繰越額から差し引く必要があります
gonnapass.com
（逆に調査で控除額の増加が認められた場合は項目[38]に計上）。
根拠例: 税務総局の公文書4943/TCT-KK（2015年）では「税務調査の処分決定で増減があった控除VAT額は、納税者は当該決定を受領した申告期間の01/GTGTにおいて項目[38]または[37]にて調整申告する」との指示が示されています
gonnapass.com
。なお、2025年以降は調査開始後の自主訂正が禁止されるため
luatvietnam.vn
、このケースでは税務当局の決定通りに[37]/[38]で処理し、必要に応じ不足税額を追納・延滞利息計上する流れとなります
gonnapass.com
。

ケース③ 還付申請に関連する控除額の補正: VAT還付申請を行った期間やその前後で誤りが見つかった場合も、項目[37]・[38]が登場します。典型例は、還付対象期間（累積還付申請を行った期間）に含まれる控除VATを事後的に調整するケースです。例えば、還付申請済みの期間に計上していた仕入税額の一部が実は控除不可であったと判明した場合、当該期間の補足申告により還付申請額を減額修正します。この結果、本来還付されるべきでなかった控除VAT分を現行期に[37]でマイナス調整し（繰越控除額の減）、誤って還付された額は納税者から返金・遅延利息支払いする必要があります
gonnapass.com
gonnapass.com
。一方、還付申請期間中に控除漏れがあり還付額を増やせる場合でも、次期以降未申告で還付請求前であれば増額分を[38]計上して繰越すことが可能です
hethongphapluat.com
（ただし既に還付請求済みの増額は認められません
hethongphapluat.com
）。
事例: ある企業Cは2014年通年で累積900百万ドンのVAT還付を申請し受領しました
gonnapass.com
。その後の調査で、2014年5月分の仕入VAT 50百万ドンが控除不適格だったと判明し、企業Cは2014年12月の還付申告を補正して還付額を850百万ドンに減額しました
gonnapass.com
。この50百万ドンは企業Cが返納すべき過大還付額となり、同時に項目[37]で当期（2015年7月）の控除繰越額を減額調整しています
gonnapass.com
。もし還付を受けず繰越していれば単なる控除放棄で済む部分ですが、一旦還付されたため返金義務が生じる点に注意が必要です。また、還付適格期間に満たなくなった場合（例：12か月未満で還付要件喪失）には、増加した納付税額を追納するとともに、本来還付できなかった控除分は[38]に計上して引き続き繰越控除に回す処理となります
gonnapass.com
。

ケース④ 投資プロジェクト・合併等による控除VATの移転: 新規の大型投資プロジェクトや企業再編（合併・事業譲渡など）の場面でも、項目[38]が使われるケースがあります。例えば、投資フェーズから事業稼働フェーズへ移行する際の未控除VATです。ベトナムでは投資段階における設備等の入力VATは累積300百万ドン以上で還付申請できますが、300百万未満の場合は還付されず持ち越されます
gonnapass.com
。この未還付の投資プロジェクトVATは、その事業が稼働を開始した期の申告書で項目[38]に計上し、通常事業の控除VATに組み入れることが認められています
gonnapass.com
。また企業合併や組織形態変更の場合、消滅会社の未控除VAT繰越額は承継会社へ引き継ぐことが可能であり、その額を承継側の申告書で項目[38]に計上する手続きとなります
thuviennhadat.vn
thuviennhadat.vn
。
根拠例: ハノイ税務局の公文書81913/CT-TTHT（2020年9月9日付）は、**「合併または企業形態変更により他社から引き継いだ未控除VAT額は、税務当局から発行される税義務承継通知書を受領した申告期間の01/GTGT申告書の『過去期控除VAT増額調整』欄（項目[38]）にて申告する」**と具体的に指示しています
thuviennhadat.vn
thuviennhadat.vn
。例えばA社がB社を吸収合併し、B社の未控除VAT残高を承継した場合、A社は税務署からの承認通知に基づき、その金額をA社の申告書で[38]に計上して以後の控除に充てることができます。なお、この処理には合併前に税務当局が被合併会社の税務状況（未控除VAT額）を調査・確認しておくことが前提となります
gonnapass.com
gonnapass.com
。

以上、実務上は**「補足申告（自主訂正）」「税務調査（行政訂正）」「還付関連」「信用承継（合併等）」**の4つが項目[37]・[38]利用の代表例となります。それぞれ法令根拠や公的文書で手続きが示されていますので、該当するケースに応じた正しい記載が求められます。

4. 項目[37]・[38]記載の具体例シミュレーション

最後に、数字を用いて項目[37]・[38]の調整が申告書に与える影響をシミュレーションします。特に、**当期納付額欄[40]・未控除VAT欄[41]・次期繰越欄[43]**への変化を確認しましょう（項目番号は01/GTGT申告書に基づく）。

例1：控除VATの減額調整（項目[37]）が必要な場合
ある企業Xは第1四半期の申告で、前期からの繰越控除VAT（項目[22]）が100百万ドン、当期の仕入VATが50百万ドン、当期の売上VATが120百万ドンだったとします。この場合、本来の計算では未控除VAT = 100 + 50 – 120 = 30百万ドンが残り、項目[41]（当期末未控除額）および項目[43]（次期繰越額）に30百万ドンが計上されます。納付額[40]は0です（控除超過のため納付税額なし）。
ところが後日、第1四半期以前の期間に控除不適格な仕入VAT 20百万ドンを誤って繰越していたことが判明しました。企業Xは補足申告によって過年度控除額を20減額し、この20百万ドンを現在の申告期の項目[37]に記載します。すると第1四半期申告書の繰越控除額は強制的に20百万ドン減額され、再計算された未控除VATは 30 – 20 = 10百万ドンとなります。つまり項目[41]および[43]は10百万ドンに修正され、項目[40]は依然0（このケースではなお控除超過のため納付なし）です。ただし将来の控除可能額が20減った分、今後の期で納付に回る可能性が高まります。項目[37]で減額調整しなければ、本来繰り越すべきでない20を誤って繰り越し続けてしまうため、当期で正しく調整することが重要です。

例2：控除VATの増額調整（項目[38]）が必要な場合
別の企業Yはある月次申告で、前月からの繰越控除VATが0、当月の仕入VATが40百万ドン、当月の売上VATが50百万ドンだったとします。この場合、未控除VATは 0 + 40 – 50 = –10百万ドンとなり（控除不足10百万ドン）、項目[40]に10百万ドンの納付税額が計上されます
meinvoice.vn
。項目[41]・[43]はいずれも0です（控除超過がないため繰越なし）。
その後、企業Yが当月に計上漏れしていた仕入VAT 15百万ドンに気付き、補足申告で追加計上しました。この15百万ドンは本来控除できたはずの額ですから、当期申告書の項目[38]に15百万ドンを記入して控除VATを増額調整します。再計算では未控除VAT = 0 + (40+15) – 50 = 5百万ドンの控除超過となり、企業Yは納付税額が0に減少（元の10百万の納付が不要）し、代わりに5百万ドンが項目[41]に未控除額として計上されます
