# 【2026年1月1日発効】ベトナム最低賃金が平均7.2%UP！あなたの給与と税金はどう変わる？日系企業・駐在員が知るべき全情報

## 新政令293/2025/ND-CPによる地域別最低賃金の具体的な変更点

2025年11月10日、ベトナム政府は労働契約に基づいて働く従業員の最低賃金を規定する**政令293/2025/ND-CP**を発行しました。この新しいルールは、**2026年1月1日**からスタートし、これまでの政令74/2024/ND-CPに代わるものなんですよ。

今回の変更で、ベトナムの地域別最低賃金がどんな風に変わるのか、具体的な数字を見ていきましょう。

### 月額最低賃金（単位: VND/月）

| 地域 | 改定前 (政令74/2024/ND-CP) | 改定後 (政令293/2025/ND-CP) | 増額 |
| :--- | :--- | :--- | :--- |
| **地域 I** | 4,960,000 VND | **5,310,000 VND** | 350,000 VND |
| 地域 II | 4,410,000 VND | **4,730,000 VND** | 320,000 VND |
| 地域 III | 3,860,000 VND | **4,140,000 VND** | 280,000 VND |
| 地域 IV | 3,450,000 VND | **3,700,000 VND** | 250,000 VND |

### 時給最低賃金（単位: VND/時）

| 地域 | 改定前 (政令74/2024/ND-CP) | 改定後 (政令293/2025/ND-CP) |
| :--- | :--- | :--- |
| 地域 I | 23,800 VND | **25,500 VND** |
| 地域 II | 21,200 VND | **22,700 VND** |
| 地域 III | 18,600 VND | **20,000 VND** |
| 地域 IV | 16,600 VND | **17,800 VND** |

今回の最低賃金引き上げは、**平均すると7.2%**の増加になるんです。これは、ベトナムで働く皆さんの生活が少しでも豊かになるように、という政府の願いが込められているのかもしれませんね。

### 地域区分の適用について

「自分の会社はどの地域に当てはまるんだろう？」って思いますよね。基本的には、皆さんの会社がある場所によって地域が決められます。もし、色々な場所に支店などがある場合は、その支店がある場所の最低賃金が適用されます。ちょっと複雑なのが、工業団地やハイテク区などにある会社の場合です。もし、そういった場所が複数の地域にまたがっている場合は、一番高い最低賃金が適用されるんですよ。シンプルに言えば、従業員にとって一番有利な条件が適用されることが多い、ということですね。

## 最低賃金改定に伴う企業が取るべき対応

この最低賃金の引き上げは、会社さんにとってはとっても大事なことです。人件費に関わることなので、早めに確認して対応を考える必要があります。日系企業の皆さんは、特に次の点に注目してみてください。

1.  **給与規定の見直し**: 今ある会社の給与規定が、新しい最低賃金のルールに合っているか確認しましょう。もし合っていなければ、変更が必要です。特に、**お仕事の内容や役職に応じたお給料が、新しい最低賃金を下回らないように調整する**ことが大切ですよ (労働法2019, 第90条)。
2.  **労働契約の確認と改定**: 従業員さんとの労働契約も、新しい最低賃金の基準に合っているか見てみましょう。もし変更が必要なら、従業員さんとしっかり話し合って、契約を直すことを検討してくださいね。
3.  **社会保険料などの計算への影響**: 最低賃金が変わると、社会保険料（健康保険、失業保険など）の計算方法も影響を受ける可能性があります。関係する法律をチェックして、正しい社会保険料を計算して納められるように準備しておきましょう。**簡単に言えば、社会保険の計算の元になるお給料が、地域別最低賃金を下回らないように、ということです。労働組合費も、社会保険の計算に使われるお給料の2%になるので、これも忘れないでくださいね。**
4.  **人件費増加による経営への影響分析と対策**: お給料が上がると、会社としては人件費が増えますよね。事前にどれくらいコストが増えるか予測して、予算計画を見直したり、業務の効率を上げる方法を考えたりすることが大切です。**給与規定や、お給料を計算するシステム（ソフトウェア）の更新も忘れずに行いましょう。**
5.  **社内での情報共有と透明性**: 新しいお給料の変更について、社内でしっかりと情報共有を行い、従業員さんに説明することも大切です。**残業代や深夜勤務手当など、他のお給料に関する制度を減らしたりしない**ことを明確に伝えることで、従業員さんの信頼も守れますよ。

## 最低賃金引き上げに伴うその他の労働関連規定への影響

最低賃金が上がると、実はお給料だけでなく、ベトナムの労働法に基づく色々な手当や補償の計算にも影響があるんです。会社さんは、次の点にも気をつけて対応してくださいね。

*   **勤務停止中の賃金**: もし、会社都合などで一時的に勤務ができなくなった場合のお給料は、会社と従業員さんの間で合意した通りに支払われます。ただ、このお給料も**地域別最低賃金を下回ってはいけない**ことになっています (労働法2019, 第99条)。
*   **損害賠償**: もし従業員さんがうっかり何か会社に損害を与えてしまった場合で、それが軽微なもので、**地域別最低賃金の10ヶ月分以下の金額**であれば、最大3ヶ月分のお給料を賠償し、毎月お給料から差し引かれることがあります (労働法2019, 第129条)。これは、万が一の時に従業員さんの負担が大きくなりすぎないように、という配慮ですね。
*   **一時的な職務変更**: 従業員さんが一時的に別のお仕事に配置される場合、新しいお仕事のお給料は、**以前のお仕事のお給料の85%以上でなければなりません。そして、これも地域別最低賃金を下回ってはいけません。**もし新しいお仕事のお給料が以前より低くても、最大30日間は以前のお給料が維持されるんですよ (労働法2019, 第29条)。これは、急な変更で従業員さんが困らないように、というルールですね。
*   **失業手当 (TCTN) & 失業保険 (BHTN)**:
    *   失業保険に支払う金額の月額上限は、地域別最低賃金の20倍と決まっています（上限額は地域によって異なり、地域Iでは106,200,000 VND、地域IVでは74,000,000 VNDまで、といった感じです）。
    *   失業手当として月に受け取れる金額は、平均的な拠出賃金の60%ですが、これも**地域別最低賃金の5倍を超えてはいけない**という上限があります（これも地域Iでは26,550,000 VND、地域IVでは18,500,000 VNDまで、といった上限があるんです）。失業した時の生活を支える大切な制度なので、こういった上限があるんですね。

## 2026年からの個人所得税（PIT）課税基準の変更

最低賃金が上がるのに合わせて、個人所得税（PIT）のルールも少し変わりますよ。**2026年1月1日**から始まる**決議110/2025/UBTVQH15**によって、個人所得税の家族控除額が変更されるんです。これは、皆さんの手取りのお給料に影響してくるので、会社さんも従業員さんにしっかり伝えてあげてくださいね。

### 家族控除額の変更

*   **納税者本人**: 月に**15,500,000 VND**（年に186,000,000 VND）
*   **扶養家族1人あたり**: 月に**6,200,000 VND**

この家族控除額は、税金を計算する際に所得から差し引かれる金額のことです。つまり、この金額が高くなると、税金がかかる基準も少し高くなる、ということですね。

### 課税所得の計算式

個人所得税がかかるお給料（課税所得）は、こんな風に計算されます。シンプルに言えば、総所得からいろいろな控除を引いた残りが課税対象、ということです。

**課税所得 = 総所得 – (強制社会保険料 + 納税者本人控除 + 扶養家族控除)**

### 具体的な課税例

具体的な例で見てみましょう。この金額を超えると個人所得税がかかり始めますよ。

*   **扶養家族がいない場合**: 控除額を引いた後の所得が月に**15,500,000 VND**以上だと、税金がかかります。
*   **扶養家族が1人いる場合**: 控除額を引いた後の所得が月に (15,500,000 VND + 6,200,000 VND) = **21,700,000 VND**以上だと、税金がかかります。
*   **扶養家族が2人いる場合**: 控除額を引いた後の所得が月に (15,500,000 VND + 6,200,000 VND × 2) = **27,900,000 VND**以上だと、税金がかかります。

### 個人所得税の対象となる給与・賃金所得

「じゃあ、どんな収入に税金がかかるの？」って疑問に思いますよね。**個人所得税法2007**（いくつか改正されていますが）のルールによると、こんなものが対象になりますよ。

*   お給料や、お給料のような性質を持つお金（ボーナスとかですね）。
*   手当や補助金（でも、功労者への手当とか、国防・治安に関する手当、危険な場所での作業手当、遠隔地手当、急な困難に対する手当、出産・養子縁組の一時金、退職手当、失業手当、社会保護手当など、一部のものは税金がかかりません）。

ちょっと複雑に聞こえるかもしれませんが、シンプルに言えば、基本的なお給料や手当は税金の対象になる、ということです。
