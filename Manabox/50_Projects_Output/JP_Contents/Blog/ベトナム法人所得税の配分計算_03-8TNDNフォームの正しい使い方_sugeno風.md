# ベトナム法人所得税の配分計算_03-8TNDNフォームの正しい使い方

こんにちは！マナラボの菅野です。

今日のテーマは「ベトナムで複数の生産拠点を持つ企業の法人所得税配分」についてです。

「うちの会社、ベトナムに工場が3つあるんだけど、税金の計算が複雑で...」というお悩み、よく聞きますよね。

実は、この記事を読むと、複数拠点の法人所得税配分がスッキリ理解できて、税務申告の不安が解消されます。

## よくある誤解：「全部まとめて計算すればいいでしょ？」

多くの企業が最初に思うのが「全部まとめて計算すればいいんじゃない？」ということです。

確かに、日本ならそうかもしれません。でも、ベトナムは違います。

ベトナムでは、各生産拠点が異なる省にまたがっている場合、法人所得税を拠点ごとに配分して納付する必要があるんです。

これをものすごくざっくりいうと、**「どこで稼いだお金は、その地域に税金を納める」**という考え方です。

## なぜ配分が必要なのか？その理由を整理してみましょう

ベトナム政府がこの制度を作った理由は、実はとてもシンプルです。

各地方政府が、自分たちの地域で稼いだお金から適切に税収を得られるようにするためです。

でも、これが企業にとっては結構面倒なんですよね。

「計算が複雑」「間違えたらどうしよう」「そもそも何をどう配分すればいいの？」

そんな不安、よく分かります。

## 03-8TNDNフォームとは？

ここで登場するのが「03-8TNDNフォーム」です。

これは、法人所得税の配分を計算するための専用フォームで、03/TNDN（法人所得税申告書）に添付して提出します。

### 03-8/TNDNフォームの構造（詳細解説版）

フォームは、大きく分けて「全体像」を記載する部分と、「各拠点への配分明細」を計算する表から構成されています。

#### 全体像のインプット項目

- **[01] 税計算期間**: 申告の対象となる事業年度を記載します。（例：2024年）
- **[02] 納税者名**: あなたの会社名を記載します。
- **[03] 税コード**: あなたの会社の税コードです。
- **[04] 総配分法人所得税額**: 全拠点で配分対象となる法人所得税の**合計額**です。
- **[05] 総配分費用**: 全拠点の配分対象となる費用の**合計額**です。これが配分割合を計算する元になります。

#### 各拠点への配分明細（表）

そして、以下の表形式で各拠点への詳細な配分を計算していきます。

| 列番号 | 内容 | 簡単な説明 |
| :--- | :--- | :--- |
| (1) | STT（連番） | 1, 2, 3... と番号を振るだけです。 |
| (2) | 指標（拠点名） | 本社、〇〇工場、××支店など、拠点の名前を記載します。 |
| (3) | 税コード/事業所コード | 各拠点の税コードや事業所コードを記載します。 |
| (4) | 郡（Huyện） | 拠点が所在する「郡」のレベルの地名です。 |
| (5) | 省（Tỉnh） | 拠点が所在する「省」のレベルの地名です。 |
| (6) | 管轄税務署 | その拠点を管轄する税務署の名前です。 |
| (7) | **生産拠点の費用** | **【計算の出発点】** 各拠点で発生した費用をここに記載します。 |
| (8) | **配分割合 (%)** | **【割合の計算】** (7) ÷ [05] で計算。費用全体のうち、この拠点が何%を占めるかを示します。 |
| (9) | **納付すべき法人税額** | **【税額の配分】** [04] × (8) で計算。この拠点が最終的に納めるべき税額です。 |
| (10) | 前期からの繰越過払税額 | もし前期にこの拠点で税金を払いすぎていれば、その額を記載します。 |
| (11) | 当期の仮払い税額 | 当期中に、この拠点で既に仮払いした税額の合計です。 |
| (12) | 差額（納付額と仮払額） | (9) - (11) で計算。単純に、納めるべき額と仮払いした額の差額を示します。 |
| (13) | **最終的な納付/還付税額** | **【最終結果】** (9) - (10) - (11) で計算。**ここがプラスなら追加納付、マイナスなら還付または繰越となります。** |

## 配分の対象となる4つのケース

実は、配分が必要なのは4つのケースだけです：

1. **電子宝くじ事業**
2. **不動産譲渡事業**  
3. **生産拠点を持つ企業**
4. **複数省にまたがる水力発電所**

多くの製造業の企業は、3番目の「生産拠点を持つ企業」に該当します。

## 配分計算の核心：費用ベースの配分

ここが一番重要なポイントです。

配分は**費用の割合**で行います。

「え？売上じゃなくて費用なの？」と思われるかもしれませんね。

実は、ベトナムの税務当局は「費用が発生した場所で事業活動が行われている」と考えるんです。

シンプルに言えば、**「その拠点でかかった費用の割合で、税金も配分する」**ということです。

## 具体的な計算方法

例えば、こんな企業があったとします：

- 本社（ハノイ）：費用 1,000万円
- 工場A（ホーチミン）：費用 2,000万円  
- 工場B（ダナン）：費用 1,000万円
- 総費用：4,000万円
- 法人所得税額：400万円

この場合の配分は：

- 本社：400万円 × (1,000万円 ÷ 4,000万円) = 100万円
- 工場A：400万円 × (2,000万円 ÷ 4,000万円) = 200万円
- 工場B：400万円 × (1,000万円 ÷ 4,000万円) = 100万円

## よくある失敗パターンとその対策

### 失敗パターン1：「優遇税制の費用も含めてしまう」

これは本当によくある間違いです。

優遇税制を受けている活動の費用は、配分計算から除外する必要があります。

なぜなら、優遇税制の活動は別途計算されるからです。

### 失敗パターン2：「仮払い税額の調整を忘れる」

法人税は年4回、予測に基づいて「仮払い」をします。でも、年末に計算した「確定税額」とは、ほとんどの場合ズレが生じます。

この**「仮払い額」と「確定額」のズレを、拠点ごとに精算するのが「調整」**です。

シンプルに言うと、「払い過ぎた分は次回に回し（繰り越し）、足りない分は追加で納付する」という作業ですね。

これを忘れると、「ある拠点では税金を滞納しているのに、別の拠点では払い過ぎ」といった、ちぐはぐな状態になってしまうので注意が必要です。

## 実務で気をつけるべきポイント

### 1. 費用の正確な把握

各拠点の費用を正確に把握することが最重要です。

「この費用はどの拠点のもの？」という判断が曖昧だと、配分も曖昧になってしまいます。

### 2. 税務署への事前相談

複雑なケースの場合は、事前に税務署に相談することをお勧めします。

「こんな配分で大丈夫ですか？」と確認しておくと、後でトラブルになるリスクを減らせます。

### 3. 書類の整理

配分計算の根拠となる書類は、しっかりと整理して保管しておきましょう。

税務調査が入った時に、すぐに説明できるようにしておくことが大切です。

## 「でも、うちの場合はどうなの？」という疑問について

「うちの会社は製造業だけど、営業所もあるし...」という疑問を持つ方もいらっしゃると思います。

実は、**営業所だけの場合は配分の対象外**です。

配分が必要なのは「生産拠点」なので、製造や加工を行っている拠点のみが対象になります。

## まとめ：配分計算は「地域貢献」の考え方

配分計算は確かに複雑ですが、ベトナム政府の「地域貢献」という考え方を理解すると、納得できる部分もあります。

各拠点で事業活動を行っている以上、その地域にも税収を還元する。

これは、長期的に事業を続けていく上で、地域との良好な関係を築くことにもつながります。

とはいえ、計算が複雑なのは事実です。

「分からないことは専門家に相談する」というのも、立派な経営判断です。

## 最後に

複数拠点の法人所得税配分は、確かに面倒な作業です。

でも、正しく理解して適切に対応すれば、税務リスクを減らすことができます。

「面倒だから後回し」ではなく、「早めに整理して安心する」という考え方で取り組んでみてください。
