flowchart TD
    A["商品の輸出入"] --> B{"国境・関門通過？"}

    B -->|No| D{"非課税地域内？"}
    D -->|Yes| E["課税対象外"]
    D -->|No| F{"現地輸出入？"}
    F -->|Yes| C
    F -->|No| G{"通過・積替？"}

    G -->|No| H{"人道支援？"}

    H -->|No| C

## 4. 納税フロー図

```mermaid
sequenceDiagram
    participant C as 商品
    participant D as 税関申告
    participant T as 課税計算
    participant P as 納税
    participant G as 通関
    
    C->>D: 商品到着
    D->>T: 申告書登録
    T->>T: 価格×税率計算
    T->>P: 納税通知
    P->>G: 納税完了
    G->>C: 通関・解放
```

## 5. 免税申請フロー

```mermaid
flowchart TD

    F -->|No| H["課税対象"]

```
