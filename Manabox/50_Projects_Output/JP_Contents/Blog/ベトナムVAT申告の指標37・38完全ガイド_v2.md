## <a name="はじめに"></a>1. はじめに：なぜ今、項目[37][38]が重要なのか？

こんにちは！マナラボの菅野です。

最近、ベトナムのVAT申告で「項目37」とか「項目38」って言葉、よく聞きませんか？「なんだかよくわからないけど、最近税理士さんがよく言ってる…」「新しいルールなの？」なんて、少し戸惑っている方もいるかもしれませんね。

大丈夫です！この記事を読み終わるころには、この謎の数字の正体がスッキリわかって、「なるほど、そういうことだったのか！」と実務で自信を持って対応できるようになりますよ。

実は、この項目たちが最近注目されているのには、大きな理由があるんです。ここ数年で、ベトナムの税務申告の「やり方」が、根本的にガラッと変わったからなんですね。

昔は、過去の申告でミスが見つかると、タイムマシンのように過去に戻って、その月の申告書自体を修正して…と、とっても大変でした。でも今は、「過去のミスは、今の申告書で調整しよう！」という、すごく合理的でスマートな方法に変わったんです。そのための専用ツールが、この項目[37]と[38]なんですね。

この記事では、この超重要ツールについて、法律の難しい話は少しだけ（笑）、実務で「あるある！」なケースや具体的な数字を使いながら、世界一わかりやすく解説していきますね。

> **ちょっと用語の話：「仕入税額控除」って？**
> これからたくさん出てくる「仕入税額控除」という言葉ですが、これはシンプルに言えば「仕入れの時に払った消費税（VAT）」のことです。日本の会計でいう「仮払消費税」とほぼ同じイメージですね。
> 国際的な文脈だと英語をそのままカタカナにして「インプットVAT」なんて言ったりもしますが、この記事では日本の皆さんにとって一番馴染みのある「仕入税額控除」という言葉で統一しますね。

## <a name="法的根拠"></a>2. 法的根拠：項目[37][38]を支える法令の全体像

「うわっ、法律の話…」って思いましたか？（笑）
大丈夫です、ここでは条文を暗記する必要は全くありません。色々な法律が、まるでRPGのパーティメンバーみたいに、それぞれどんな役割を担っているのか、キャラクター紹介風に見ていきましょう！

*   **リーダー（税務管理法）**:
    パーティの頼れるリーダーです。「申告で間違えちゃっても、後からちゃんと直せば大丈夫だよ」という、基本的なルールを決めてくれている存在です。これが全ての基本になります。

*   **革命家（政令126/2020/NĐ-CP）**:
    今回の物語の最重要人物です！「昔の間違いを直すのに、いちいちタイムマシンに乗らなくていいよ！**今の申告書でサクッと調整しちゃおう！**」という、画期的な新ルールを編み出した革命家なんですね。項目[37][38]が脚光を浴びたのは、まさにこの人のおかげです。

*   **専門家（2024年改正VAT法）**:
    VAT（付加価値税）のことなら何でも知っている専門家です。「そもそも、どんなものが仕入税額控除の対象になるの？」といった、VATそのものの中身に関するルールを決めています。

*   **案内人（補足申告書 01/KHBS）**:
    過去のミスを修正するときに、「どこがどう間違っていて、結果的に控除額がいくら増減するのか」を税務署に説明するための公式な書類です。この案内人が計算した結果を、項目[37]や[38]に書き写す、という流れになります。

*   **サポート役（公式文書いろいろ）**:
    税務署が出してくれる、いわば「公式Q&A集」です。「こういう場合はどうすればいいの？」という細かい疑問に答えてくれる、頼れるサポートメンバーですね。

特に重要なサポート役（公式文書）たちの結論を、表にまとめてみました。これを見ると、具体的なシーンがイメージしやすくなりますよ。

| 公式文書名                               | この一文でわかる！結論                                                     |
| :--------------------------------------- | :------------------------------------------------------------------------- |
| **税務総局 公文書 4943/TCT-KK**          | 税務調査で指摘された調整は、税務署の「決定書」通りに項目[37]・[38]に書く。  |

シンプルに言えば、**「革命家（政令126号）が生み出した『今の申告書で調整する』という画期的な方法を、他のメンバーたちがそれぞれの立場でサポートしている」**、そんなイメージです。

## <a name="基本ルール"></a>3. 基本ルール：項目[37][38]の正しい記載方法

さて、登場人物がわかったところで、実際に項目[37]と[38]を使うときの、3つの大事な「合言葉」を覚えておきましょう！

*   **合言葉①：「仕入だけ！」**
    これは、私たちが何かを買ったときにかかったVAT（仕入税額控除）の調整専用です。売上に関するVATは管轄外なので、間違えないようにしたいですね。

*   **合言葉②：「増えたら[38]、減ったら[37]」**
    シンプルですよね！過去に計上し忘れてたVATがあって、控除できる金額が**増える**なら**項目[38]**。逆に、間違って多く計上しちゃってたのを**減らす**なら**項目[37]**です。なんだか、[38]の方が数字が大きいからプラスな感じがしませんか？（笑）

*   **合言葉③：「直すのは『今』！」**
    これが革命家、政令126号の魔法でしたね。去年の5月のミスだとしても、去年の申告書を引っ張り出してくるんじゃなくて、**今まさに作っている申告書**で調整するんです。これ、本当に実務が楽になりました。

この3つの合言葉さえ覚えておけば、基本はバッチリです！

## <a name="4大ケース"></a>4. 実務でよくある4大ケースと具体的な使い方

理屈はわかったけど、じゃあ具体的にどんなときに使うの？って思いますよね。わかります。
ここからは、実務で本当によく出会う「あるある！」な4つのケースを見ていきましょう。

### <a name="ケース1"></a>ケース①：「あ、忘れてた！」自主的な修正ケース

一番よくあるのが、この「うっかりさん」ケースです。
「あ！この経費のインボイス、先月の申告に入れ忘れてた！」とか、逆に「あれ、このインボイス、控除しちゃダメなやつだった…」なんてこと、ありますよね。

大丈夫、そんなときこそ項目[37][38]の出番です。

*   **手順はかんたん**:
    1.  まず「ごめんなさい、ここを間違えました」という反省文、もとい「補足申告書（01/KHBS）」を作ります。
    2.  そこで計算された「増える控除額」または「減る控除額」を、**今の申告書**の項目[38]（増える場合）か[37]（減る場合）に書くだけです。

これでスマートに修正完了です！

### <a name="ケース2"></a>ケース②：「税務調査で指摘されちゃった…」ケース

次は、税務署の調査で「ここ、間違ってますよー」と指摘されちゃった場合です。ちょっとドキッとしますよね。
でも、修正方法は基本的に同じなので安心してください。

*   **手順のちがい**:
    この場合、私たちの方で「補足申告書」を作る必要はありません。税務署から「決定書」という公式な通知が来ます。
    私たちは、その**決定書に書いてある金額**を、通知を受け取った月の申告書の項目[37]（大体は減額ですよね…）に記入すればOKです。

税務署の指示に従う、というところがポイントですね。

### <a name="ケース3"></a>ケース③：「VAT還付が絡むと、ちょっと複雑」ケース

VATの還付（払いすぎた税金を返してもらうこと）を申請した後に、その期間の申告にミスが見つかると、話が少しだけ複雑になります。
なぜなら、これは単なる書類上の数字の修正じゃなくて、「実際のお金の動き」が絡んでくるからです。

*   **還付してもらいすぎた場合**:
    これは大変です！
    1.  まず、多くもらいすぎた還付金を国に**返金**します。もちろん、延滞税というペナルティも付きます…。
    2.  それと同時に、今の申告書の**項目[37]**で、繰越しの控除額もちゃんと減らしておきます。
    お金を返す手続きと、申告書を直す手続き、両方が必要になるんですね。

*   **還付後に「計上漏れ」を発見した場合**:
    残念ながら、「すみません、これも控除し忘れてたので、追加で還付してください！」ということはできません。一度終わったお祭りは終わり、という感じです。
    でも、その控除する権利が消えるわけではありません。**今の申告書**の**項目[38]**に計上して、将来払う税金と相殺するために使うことができます。諦めないでくださいね！

### <a name="ケース4"></a>ケース④：「会社が合体！」企業再編ケース

会社が合併したりすると、消える会社が持っていた「繰越控除VAT」という、いわば「税金の割引クーポン」を、新しい会社が引き継ぐことができます。

*   **手順のポイント**:
    これは当事者だけで勝手にできるものではなくて、税務署にきちんと認めてもらう必要があります。
    税務署のチェックを受けて、「はい、この金額を引き継いでOKですよ」というお墨付き（税義務承継通知書）をもらいます。
    そして、その**通知書に書かれた金額**を、新しい会社が**項目[38]**に記入して、ありがたくクーポンを引き継ぐ、という流れになります。

## <a name="シミュレーション"></a>5. 具体例シミュレーション：申告書への影響を数字で確認

百聞は一見にしかず、ですよね。
ここからは電卓片手に、実際の申告書がどう変わるのか、数字で見ていきましょう。ちょっとだけ数字が出てきますが、ゆっくり説明するので安心してくださいね。（項目番号は実際の申告書フォーム01/GTGTのものです）

---

### 例1：「しまった、控除しすぎてた！」減額調整（項目[37]）
**シナリオ**: A社は、第2四半期（4月～6月）のVAT申告の準備をしています。まずは、手元にある数字を整理してみましょう。

*   **前提となる数字**:
    *   前期（第1四半期末）から繰り越されてきた、まだ使っていない仕入税額控除（割引クーポン）が **100** ありました。 `[項目22]`
    *   そして、この第2四半期中に仕入れなどで発生した、控除できる仕入税額（新しくゲットした割引クーポン）が **50** です。 `[項目24, 25]`
    *   一方、第2四半期の売上から発生した、国に納めるべき売上VATは **120** でした。 `[項目32, 33]`

**ミスの発見**:
ここで経理担当者は気づきます。「しまった！前期から繰り越してきたクーポン100の中に、本当は控除できないVATが**20**も混じっていた！」

*   **もし、このミスに気づかなかったら…（調整前の計算）**:
    *   **計算**: (前期のクーポン100 + 今期のクーポン50) - 今期の売上VAT120 = **30**
    *   → このままだと、納税は0で、次期に**30**の割引クーポンが繰り越される計算になります。でも、これでは間違いを未来に持ち越してしまいますよね。

*   **正しい調整をしよう！**:
    1. 「ごめんなさい」の補足申告書を作ります。
    2. **今の第2四半期申告書**の**項目[37]に「20」と記入**します。

*   **調整後の正しい申告**:
    *   前期からの割引クーポン残高 `[22]`：100
    *   今期ゲットした割引クーポン `[24][25]`：50
    *   **間違いを減らす調整 `[37]`**: **20**
    *   今期払うべき税金 `[32][33]`：120
    *   **再計算**: (100 + 50 - **20**) - 120 = **10**
    *   **結果**: 見てください！納税額は0のままですが、**未来の自分が助かる「割引クーポン残高 `[43]`」が、ちゃんと正しい「10」になりました**。これで将来余計な税金を払わなくて済みますね。ファインプレーです！

---

### 例2：「ラッキー！控除漏れ発見！」増額調整（項目[38]）
**シナリオ**: B社は7月分の月次申告を終えて、10百万VNDを納税しました。ホッとしたのも束の間、「あ！7月分のインボイスで、15百万VNDも計上し忘れてる！」と気づきました。

*   **調整前の状況**:
    *   7月は10百万VNDの税金を**すでに払ってしまいました**。

*   **正しい調整をしよう！**:
    1. 「これ、忘れてました！」の補足申告書を作ります。
    2. **気づいた月（例えば9月）の申告書**の**項目[38]に「15」と記入**します。

*   **調整後のハッピーな影響**:
    *   まず、9月の申告で**15百万VND分の割引クーポンが復活**します。これで9月以降の納税が楽になりますね。
    *   さらに！7月は本当なら税金を払う必要がなかった（むしろ5のクーポンが残る計算だった）ことが確定したので、**払ってしまった10百万VNDは「払いすぎ」**ということになります。このお金は、将来の税金と相殺したり、返金してもらったりできます。キャッシュフローが大きく改善しますね！

---

### 例3：「還付金、もらいすぎてました…」一番大変なケース
**シナリオ**: C社は、900百万VNDもの大金！をVAT還付で受け取りました。しかしその後、還付の対象期間に50百万VNDの控除できないVATが混じっていたことが発覚…。

*   **やるべきこと**:
    1.  まず、税務署に「すみません、計算間違ってました！」と補足申告します。
    2.  次に、一番つらいですが、**もらいすぎた50百万VNDを国に返金します**。もちろん、**延滞税というペナルティ付き**です…。
    3.  最後に、**今の申告書**の**項目[37]に「50」と記入**して、クーポン残高も正しく直します。

*   **教訓**: お金が動いた後の修正は、ペナルティもあって本当に大変です。還付申請は、特に慎重にやりたいですね。

## <a name="まとめ"></a>7. まとめ：怖がらずに使ってみよう！

さて、今日の冒険をまとめますね！
ベトナムVAT申告書の項目[37][38]は、**「過去の仕入税額控除のミスを、今の申告書でスマートに調整する」**ための、とっても便利で重要なツールでした。

**実務担当者のみなさんへ、3つの応援メッセージ:**
*   **根拠を確認しよう**: [37][38]に数字を書くときは、その数字、どこから来たんだっけ？と一度立ち止まって、補足申告書などと合っているか確認しましょう。
*   **未来を予測しよう**: この調整をしたら、納税額やクーポン残高はどう変わるかな？と、ちょっとだけ未来をシミュレーションしてみると、経理の仕事がもっと楽しく、プロフェッショナルになりますよ。
*   **期限は守ろう**: 税務署との約束（期限）は、守ると信頼関係につながります。特に還付金の返還などは、遅れないようにしたいですね。

最初は少しとっつきにくいルールかもしれませんが、一度理解してしまえば、これほど頼りになるツールはありません。怖がらずに、正しく活用して、日々の業務をもっとスムーズに、そして正確に進めていきましょう！

## <a name="関連法令"></a>8. もっと知りたい方へ：関連法令リスト

もっと深く知りたい！という探求心あふれる方のために、今回の話の根拠になった法律たちをリストアップしておきますね。困ったときのお守り代わりにもどうぞ。

### 法律 (Law / Luật)
*   **税務管理法 No. 38/2019/QH14**: 「間違えても、後から直せるよ」の基本ルール。
*   **付加価値税法 (2024年改正)**: 「そもそもVATって何？」のルール。
