# ベトナムで出資持分を売却したら個人所得税の申告が必要です｜フォーム04/CNV-TNCNの書き方を優しく解説

こんにちは！マナラボの菅野です。

今日は、ベトナムでビジネスをされている方、特に**現地法人に出資している方**にとって、いつか必ず直面する可能性がある「資本譲渡の税務申告」についてお話しします。

最近、こんなご相談をいただきました。

> 「ベトナムの会社に投資していた持分を売却したんですが、税務申告が必要だと聞きました。フォーム04/CNV-TNCNというのを出すらしいんですが、何をどう書けばいいのか全然わからなくて...」

わかります。

ベトナムの税務手続きって、日本と全然違うんですよね。しかも英語じゃなくてベトナム語の書類だったりして、「これ、どこに何を書くの？」って途方に暮れる気持ち、すごくよくわかります。

## この記事を読むとこんなメリットがあります

- フォーム04/CNV-TNCNが何なのか、何のための書類なのかがわかります
- どの欄に何を書けばいいのか、具体的にイメージできます
- 提出する書類が何か、いつまでに出せばいいのかが明確になります
- よくある失敗やミスを事前に避けられます

難しそうに見えますが、ひとつずつ理解していけば大丈夫です。

一緒に見ていきましょう。

## そもそもフォーム04/CNV-TNCNって何？

### これは「資本を譲渡したときの個人所得税申告書」です

シンプルに言えば、**株や出資持分を売って利益が出たら、その利益に税金がかかるので申告してね**、という書類です。

日本でも株を売ったら譲渡所得税を払いますよね。あれのベトナム版だと思ってください。

ただ、日本と大きく違うのは、**年間でまとめて申告するのではなく、売買のたびに申告する**という点です。

これ、けっこう重要です。

### このフォームの法的根拠

このフォームは、以下の法令に基づいて定められています。

**主な根拠法令：**
- **通達80/2021/TT-BTC**（2021年9月29日公布）付録II
  - 税務管理法の実施ガイダンスで、申告書の様式を定めています
- **通達40/2025/TT-BTC**（2025年6月30日公布）第1条第7項g号
  - 通達80/2021を修正・補完し、2025年8月14日から施行されています

法令の話って堅苦しいんですが、実務では「どの法令に基づいているのか」を知っておくと、税務署とのやり取りで役立つことがあります。

### 誰がこのフォームを使うの？

以下のような方が対象になります。

**1. ベトナム法人への出資持分を売却した人**
- 有限責任会社や株式会社への出資を売った
- 持分の一部でも全部でも、売ったらアウトです

**2. 証券を売却した人（特定の場合）**
- 証券会社で源泉徴収（0.1%）されなかった取引
- 直接、税務署に申告が必要なケース

**根拠条文：**
- **通達111/2013/TT-BTC** 第25条第1項d号
- **政令12/2015/ND-CP** 第2条第16項2号

証券会社が源泉徴収してくれる場合は、個別申告は不要です。これは通達で明確に定められています。

**3. 複数人分をまとめて申告する組織や税理士**
- 代理で申告する場合も同じフォームを使います

### 実務で注意したいポイント

ベトナムでは**「取引ごとに申告」**というルールなので、売買契約を結んだら速やかに動き始める必要があります。

「後でまとめてやればいいや」が通用しないんですね。

この「取引ごと申告」の原則は、**個人所得税法2007（Law 04/2007/QH12）第7条**および**2012年改正法（Law 26/2012/QH13）第1条第3項**で定められています。

特に以下のケースは複雑になりやすいので、早めに専門家に相談することをおすすめします。

- 出資持分を現物（不動産や設備など）で受け取った場合
- 会社が解散して持分が払い戻される場合
- 外国人が譲渡する場合（租税条約が使えるかもしれません）

## 税金はいくらかかるの？税率と計算方法

### 出資持分を売った場合：税率20%

まず、**課税所得**を計算します。

```
課税所得 ＝ 売った金額 − 当初出資した金額 − 売るためにかかった費用
```

たとえばこんな感じです。

**【計算例】**
- 売却価格：500,000,000 VND（だいたい500万円くらい）
- 当初の出資額：200,000,000 VND
- 仲介手数料など：10,000,000 VND

```
課税所得 = 500,000,000 − 200,000,000 − 10,000,000 = 290,000,000 VND

納税額 = 290,000,000 × 20% = 58,000,000 VND（約58万円）
```

**売却益に対して20%の税金**がかかるイメージです。

#### 税率の法的根拠

この20%という税率は、以下の法令で定められています。

- **個人所得税法2007**（Law 04/2007/QH12）
- **個人所得税法2012年改正法**（Law 26/2012/QH13）第1条第3項

法律でしっかり決まっているので、税務署の担当者によって変わることはありません。

### 証券を売った場合：税率0.1%

証券の場合は計算がシンプルです。

```
納税額 ＝ 売却金額 × 0.1%
```

ただし、これは証券会社で源泉徴収されていない場合の話です。

通常の証券取引であれば、証券会社が0.1%を天引きしてくれるので、個別に申告する必要はありません。

**根拠条文：**
- **通達111/2013/TT-BTC** 第25条第1項d号
- **政令12/2015/ND-CP** 第2条第16項2号

### 租税条約を使うと税金が安くなることも

もしあなたが日本人で、ベトナムに183日未満しか滞在していない場合、日本とベトナムの租税条約を使って税金を減らせる可能性があります。

簡単に言えば、「二重に税金を取られないようにする仕組み」ですね。

ただし、これには居住者証明書など、別途書類が必要になります。詳しくは税理士さんに相談するのが確実です。

## フォーム04/CNV-TNCNの書き方を一緒に見ていきましょう

さて、ここからが本題です。

このフォーム、大きく4つのセクションに分かれています。順番に見ていきましょう。

### セクション1：基本情報を書く欄

まずは「いつ、何回目の申告か」を記入します。

**[01a] 発生日**
→ 譲渡契約を結んだ日付を書きます。

**[01b] 月・四半期・年**
→ 通常は空欄でOKです。

**[02] 初回申告**
→ この取引について初めて申告するなら、ここにチェック☑

**[03] 追加申告**
→ 修正や追加で出す場合は、何回目かを書きます。

#### 特殊なケースでチェックを入れる欄

以下に該当する場合は、チェックを入れます。

- ☑ 租税条約による減免を受けたい
- ☑ 複数人分をまとめて申告する（会社が代理申告する場合など）

### セクション2：売った人（納税者）の情報

ここはあなた自身の情報を書く欄です。

| 項目 | 何を書くか |
|------|----------|
| [04] 氏名 | パスポートに書いてある通りの名前 |
| [05] 納税者番号（TIN） | ベトナムの税務登録番号（持ってなければ空欄でもOK） |
| [06] 住所 | 今住んでいる住所を詳しく |
| [07] 区・郡 | 住所の行政区分 |
| [08] 省・市 | ハノイ、ホーチミンなど |
| [09] 電話番号 | 連絡がつく番号 |
| [11] Email | 税務署からの連絡用 |

#### 代理で申告してもらう場合

税理士さんや会社に代理申告してもらう場合は、[12]以降に代理人の情報を書きます。

### セクション3：買った人（譲受人）の情報

持分を買ってくれた相手の情報です。

複数の人に分けて売った場合は、それぞれの情報を表形式で書きます。

| 項目 | 何を書くか |
|------|----------|
| [23] 順番 | 1、2、3... |
| [24] 氏名 | 買った人の名前 |
| [25] 税務登録番号 | 相手のTIN |
| [26] 身分証明書 | パスポートや国民IDの番号 |
| [27] 契約番号 | 譲渡契約書の番号 |
| [28] 契約日 | 契約を結んだ日 |

### セクション4：会社の情報

あなたが持分を持っていた会社の情報です。

| 項目 | 何を書くか |
|------|----------|
| [29] 会社名 | 正式な会社名 |
| [30] 税務登録番号 | 会社のTIN |
| [31] 住所 | 会社の登記住所 |

この辺は、会社の登記書類を見れば全部書いてあるはずです。

### セクション5：税額の計算（一番大事なところ）

ここが一番重要です。

でも、落ち着いて計算すれば大丈夫です。

#### 出資持分を売った場合

| 項目 | コード | 何を書くか |
|------|-------|----------|
| 譲渡価額 | [37] | いくらで売ったか（VND） |
| 取得原価 | [38] | 当初いくら出資したか（VND） |
| 譲渡費用 | [39] | 仲介手数料などの経費（VND） |
| 課税所得 | [40] | [37] − [38] − [39] |
| 納税額 | [41] | [40] × 20% |
| 租税条約による減免額 | [42] | 減免がある場合のみ |
| 最終納税額 | [43] | [41] − [42] |

**ポイント：取得原価と譲渡費用は、きちんと証明できる書類を用意しておく**

これがないと、税務署に「証明できないなら取得原価ゼロとみなします」と言われて、売却価格全額に税金がかかる可能性があります。

怖いですよね。

だから、当初の出資契約書や、会社の会計帳簿のコピーなど、証拠をしっかり準備しておくことが大切です。

#### 証券を売った場合

| 項目 | コード | 何を書くか |
|------|-------|----------|
| 譲渡価額 | [44] | いくらで売ったか（VND） |
| 納税額 | [45] | [44] × 0.1% |
| 租税条約による減免額 | [46] | 減免がある場合のみ |
| 最終納税額 | [47] | [45] − [46] |

証券はシンプルですね。

### 最後に署名・捺印

書類の一番下に署名する欄があります。

- 署名日
- 署名者の氏名
- 役職（会社が代理申告する場合）
- 会社印（法人の場合）

自分で申告する場合は、自分の名前を書いて署名すればOKです。

## 一緒に提出する書類は何？

申告書だけ出せばいいわけじゃないんです。

証拠書類も一緒に出す必要があります。

### 必要書類の法的根拠

提出書類の規定は、**政令126/2020/ND-CP 付録I セクション9.4**に明確に定められています。

以下の書類が必須です。

### 必ず必要な書類

| 書類 | 形式 | 注意点 |
|------|------|--------|
| フォーム04/CNV-TNCN | 原本 | 署名・捺印を忘れずに |
| 譲渡契約書のコピー | コピー | 全ページ |
| 取得原価の証明（出資契約書など） | コピー | 当初出資した時の書類 |
| 譲渡費用の証明（領収書など） | コピー | 仲介手数料の請求書など |

**補足：** 現物出資で取得した場合は、出資契約書の代わりに現物出資時の評価証明書が必要です（政令126/2020/ND-CP 付録I セクション9.4）。

### 場合によって必要な書類

| 書類 | いつ必要か | 根拠法令 |
|------|----------|---------|
| 付録04-1/CNV-TNCN | 複数人分をまとめて申告する場合 | 通達80/2021/TT-BTC |
| 居住者証明書 | 租税条約を使う場合 | 政令126/2020/ND-CP |
| 租税条約適用申請書 | 租税条約を使う場合 | 政令126/2020/ND-CP |

### 提出前のチェックリスト

書類を出す前に、以下を確認しましょう。

- [ ] すべての書類がそろっているか
- [ ] コピーは読みやすいか（薄すぎないか）
- [ ] 英語や日本語の書類は、ベトナム語翻訳をつけたか
- [ ] 契約書に当事者全員の署名があるか
- [ ] 金額の記載に誤りがないか（桁数の確認）
- [ ] 税務登録番号（TIN）は正しいか
- [ ] 連絡先は正確か

特に**金額の桁数ミス**は多いです。

ベトナムドンは桁が大きいので、ゼロの数を間違えやすいんですよね。注意しましょう。

## いつまでに申告すればいいの？

### 申告期限：契約締結日から10日以内

これ、けっこう短いです。

たとえば、3月15日に契約を結んだら、3月25日までに申告書を出さないといけません。

土日祝日も含めて10日なので、実質的にはもっと短いと考えた方がいいです。

**根拠条文：** 政令126/2020/ND-CP 付録I セクション9.4

この期限は法令で明確に定められているので、「知らなかった」では済まされません。

### 納税期限：申告書を出してから10日以内

申告と納税は別です。

申告書を提出したら、税務署から納付書をもらって、それを銀行に持っていって納税します。

これも10日以内なので、合計すると契約から20日以内には全部終わらせる必要があります。

けっこう忙しいですよね。

### 遅れるとどうなるの？

遅延すると、延滞金がかかります。

| 遅れた内容 | ペナルティ | 根拠法令 |
|-----------|-----------|---------|
| 申告が遅れた | 本税額 × 0.03% × 遅延日数 | 税務管理法 |
| 納税が遅れた | 本税額 × 0.05% × 遅延日数 | 税務管理法 |
| 申告漏れ | 本税額 × 10〜20% の過少申告加算税 | 税務管理法 |
| 虚偽申告 | 本税額 × 20〜50% の重加算税 + 刑事罰の可能性 | 税務管理法 |

たとえば、納税額が5,000万VND（約50万円）で、30日遅れたとします。

```
延滞金 = 50,000,000 × 0.05% × 30日 = 750,000 VND（約7,500円）
```

少額に見えますが、遅れれば遅れるほど増えていくので、できるだけ期限を守りましょう。

## よくある質問にお答えします

実務でよく聞かれる質問をまとめました。

### Q1. 税務登録番号（TIN）を持っていないんですが...

A. 大丈夫です。まずTINを取得する手続きをしましょう。

最寄りの税務署に以下を持っていけば、5〜10営業日くらいで発行してもらえます。

- パスポートのコピー
- 住所証明（賃貸契約書など）
- 申請書（税務署でもらえます）

もし時間がない場合は、TIN申請中でも申告書は出せます。その場合、TIN欄に「申請中」と書いておきましょう。

### Q2. 当初いくら出資したか、証明する書類がないんですが...

A. これは困ります...が、代替手段があります。

**政令126/2020/ND-CP 付録I セクション9.4**では、以下の書類が取得原価の証明として認められています。

- 会社の定款（出資金額が書いてあることが多い）
- 会社の株主名簿
- 会社が発行する出資証明書
- 銀行の送金記録（出資した時の記録）
- 会計帳簿の該当ページ

それでもダメな場合、税務署によっては「売却価格の50%を取得原価とみなす」という取り扱いをしてくれることもあります。

ただし、これは税務署ごとに対応が違うので、事前に相談しておくのが安全です。

### Q3. 外国人ですが、同じフォームを使いますか？

A. はい、日本人でもベトナム人でも、同じフォームを使います。

**個人所得税法2007（Law 04/2007/QH12）第7条**では、居住者・非居住者を問わず、資本譲渡所得は「取引ごとに申告」と定められています。

ただし、あなたがベトナムに183日未満しか滞在していない場合、租税条約を使って税金を減らせる可能性があります。

その場合は、日本の税務署から「居住者証明書」を取り寄せる必要があります。

### Q4. 持分を現物（不動産など）で受け取った場合は？

A. 現物の市場価格で評価して、それを譲渡価額とします。

**通達111/2013/TT-BTC 第14条**では、現物で譲渡対価を受け取った場合、「市場における当該資産の販売価格」で評価すると定められています。

この場合、独立した鑑定人に鑑定してもらい、鑑定書を申告書に添付するのが安全です。

### Q5. 持分を売って損失が出た場合はどうなりますか？

A. 損失が出た場合、納税義務はありません。

ただし、残念ながらベトナムでは損失を他の所得から差し引いたり、翌年に繰り越したりすることはできません。

**個人所得税法2007（Law 04/2007/QH12）**では、資本譲渡所得は他の所得と分離して課税されるため、損益通算の制度がないんです。

シンプルに「損失が出たら税金ゼロ」というだけです。

### Q6. 複数回に分けて譲渡する場合は？

A. 売るたびに申告が必要です。

**個人所得税法2007（Law 04/2007/QH12）第7条**および**2012年改正法（Law 26/2012/QH13）第1条第3項**で、資本譲渡所得の課税期間は「取引ごと」と明確に定められています。

たとえば、3月に半分売って、6月に残り半分を売ったら、それぞれ申告書を出します。

- 第1回譲渡：3月15日 → 3月25日までに申告
- 第2回譲渡：6月20日 → 6月30日までに申告

面倒ですが、ベトナムのルールなので仕方ありません。

### Q7. 会社清算に伴う持分の払い戻しは課税されますか？

A. はい、原則として課税対象です。

**通達78/2014/TT-BTC 第14条**では、企業の清算に伴う資本の払い戻しも「資本譲渡」に該当すると定められています。

**計算方法：**
```
課税所得 = 払戻額 − 当初出資額
```

ただし、以下の場合は非課税となります。
- 会社の破産により払戻額が出資額を下回る場合
- 法令に基づく強制清算の場合（破産法に基づく手続きであることの証明が必要）

### Q8. 租税条約の適用を受けるための手続きを教えてください。

A. 以下の書類を申告時に併せて提出します。

**必要書類：**
1. 居住者証明書（Certificate of Residence）
   - 本国税務当局が発行
   - 有効期限内のもの（**通達40/2025/TT-BTC**により、2025年8月14日以降は有効期限2年に延長）
   
2. 租税条約適用申請書（ベトナム財務省指定様式）

3. 減免計算書（軽減税率を適用する旨の計算書）

**提出先：** 管轄税務署の国際税務課

**処理期間：** 約30〜60日（**通達40/2025/TT-BTC**により事前申請制度が導入され、従来より短縮されました）

## 2025年の改正で何が変わったの？

最後に、最新情報をお伝えします。

### 通達40/2025/TT-BTCによる主な変更点

**公布日：** 2025年6月30日

**施行日：** 2025年8月14日

この改正で、いくつかの変更がありました。

#### 変更点1：申告書の様式が少し簡素化

記入欄が一部減って、書きやすくなりました。

また、電子申告に対応した項目が追加されています。

**根拠：** 通達40/2025/TT-BTC 第1条第7項g号

#### 変更点2：添付書類の範囲が明確化

「取得原価の証明書類って具体的に何？」という疑問が多かったので、法令で明示されました。

これにより、会計帳簿のコピーや銀行送金記録も正式に認められることが明確になりました。

**根拠：** 政令126/2020/ND-CP 付録I セクション9.4（通達40/2025により明確化）

#### 変更点3：租税条約適用手続きの簡素化

居住者証明書の有効期限が**1年から2年に延長**されました。

これまでは毎年取り直す必要があったんですが、2年に1回で済むようになったので、少し楽になりましたね。

また、事前申請制度が導入されて、処理期間も短くなりました。

**根拠：** 通達40/2025/TT-BTC 第1条第7項

### いつから新しい様式を使えばいいの？

**2025年8月14日以降の取引**については、新様式を使います。

それより前の取引については、旧様式でも受け付けてもらえます。

**経過措置：** 通達40/2025/TT-BTC 第2条

## まとめ：押さえておきたい5つのポイント

長くなったので、最後にまとめますね。

### 1. 申告期限は契約締結から10日以内

これが一番重要です。

契約を結んだら、すぐに動き始めましょう。

**根拠：** 政令126/2020/ND-CP 付録I セクション9.4

### 2. 取得原価の証明書類は必ず準備する

これがないと、税金が高くなります。

当初の出資契約書や会計帳簿のコピーを確保しておきましょう。

**根拠：** 政令126/2020/ND-CP 付録I セクション9.4

### 3. 租税条約を使えるかチェックする

日本人で、ベトナムに短期滞在の場合、税金を減らせるかもしれません。

事前に専門家に相談しておくと安心です。

### 4. 申告と納税は別の期限

申告しただけで終わりじゃありません。

納税まで完了させないと、延滞金がかかります。

### 5. わからなければ専門家に頼る

以下のような複雑なケースは、無理せず税理士さんに相談しましょう。

- 租税条約を使いたい
- 複数の人に分割して売る
- 現物で受け取る
- 過去に未申告がある
