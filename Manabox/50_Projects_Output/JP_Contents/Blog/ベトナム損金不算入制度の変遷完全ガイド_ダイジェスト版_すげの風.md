# ベトナム法人税「損金不算入費用」の変遷完全ガイド ダイジェスト版（2008年～2025年）

こんにちは！マナラボの菅野です。

ベトナムで事業をしている皆さん、法人税の「損金不算入費用」について「昔と今で全然違うって聞くけど、どうすればいいの？」という疑問、よく聞きます。実は、ベトナムの損金不算入制度は2008年から2025年までに4回もの大きな改正を経て、その姿を大きく変えてきたんですよ。過去の常識だけにとらわれていると、知らないうちに損をしてしまったり、税務上のリスクを抱えてしまう可能性もあるんです。

この記事では、そんなベトナム損金不算入制度の17年間の変遷を、忙しい皆さんにもサッと理解してもらえるよう、ダイジェストでお届けします。これだけ読めば、今のルールがなぜこうなっているのか、そしてこれからどうなるのかがバッチリ分かりますよ！

## 超ざっくり！ベトナム損金不算入制度の歴史

| 期間 | 主要な法令 | ざっくりポイント |
| :--- | :--- | :--- |
| **第1期：黎明期**<br/>(2008年～2014年) | 法律14/2008/QH12号<br/>通達130/2008/TT-BTC号 | **「厳格な制限」**が中心。特に広告宣伝費に10%上限があり、福利厚生費も原則不算入と、かなり厳しかったんです。 |
| **第2期：劇的緩和期**<br/>(2015年～2016年) | 法律71/2014/QH13号<br/>通達96/2015/TT-BTC号 | **「企業活動の促進」**へ大転換！広告宣伝費の上限が撤廃され、福利厚生費も一部損金算入可能に。企業が大喜びした改正でした。 |
| **第3期：国際対応期**<br/>(2017年～2024年) | 政令20/2017/NĐ-CP号<br/>通達25/2018/TT-BTC号 | **「国際基準と微調整」**の時代。多国籍企業の租税回避（BEPS）対策として関連者間利息に制限が導入されたり、従業員向け保険料の上限が引き上げられたりしました。 |
| **第4期：新時代へ**<br/>(2025年～) | 新法人所得税法<br/>67/2025/QH15号 | **「法典化とコンプライアンス重視」**。ルールがより明確になり、環境対策や社会貢献への優遇も盛り込まれる、まさに未来志向の税制です。 |

## 各期のポイントを深掘り！

### 第1期：厳格だった黎明期（2008年～2014年）- 「ガチガチのルールに悩んだ時代」

この時期は、まだベトナムの税制が整い始めた頃で、企業にとってはかなり厳格な制限が多かったです。特に悩ましかったのは「広告宣伝費の上限規制」でしたね。年間総損金算入費用の**10%**（新規設立後3年間は15%）を超える部分は、残念ながら損金にできませんでした。マーケティングに力を入れたいのに、税金が増えてしまう...というジレンマを抱える企業が多かったんです。

また、従業員向けの福利厚生費も原則損金不算入で、「社員のために何かしたい」と思っても税務上の制約が大きく、なかなか実現しづらい時代でした。人件費や固定資産の減価償却費についても、かなり細かく厳格なルールがあったんですよ。当時の政府は、税収確保と企業の適正経営を重視していたため、このような厳しめのルールが敷かれていたんですね。

### 第2期：劇的な緩和期（2015年～2016年）- 「企業の自由度が爆上がり！」

2015年は、ベトナムの法人税制にとってまさに「歴史的な転換点」となりました。この時期の改正は、企業の経済活動を促進するための規制緩和が核心でした。

最大の変化は、やはり**広告宣伝費の上限撤廃**でしょう！これまで企業活動の足かせとなっていた10%上限が完全に撤廃され、企業は必要なマーケティング費用を金額の制限なく損金算入できるようになったんです。これで、もっと自由にビジネスを展開できるようになったわけですね。

さらに、従業員やそのご家族のための福利厚生費も、**年間の実績給与総額の平均1ヶ月分**を上限として損金算入が明確に認められるようになりました。これは、従業員の待遇改善を税務面から後押しする、本当に嬉しい改正でした。従業員向け任意保険料の損金算入も認められ、会社の福利厚生を充実させやすくなったんです。

この劇的な緩和により、企業の税務戦略の自由度は格段に向上し、ベトナムビジネスは「攻めの時代」へと突入しました。

### 第3期：国際基準への対応と微調整（2017年～2024年）- 「世界に合わせた税制へ」

2015年の大改正後、ベトナムの税制は国際的な流れに合わせて調整されていきました。特に注目すべきは、**関連者間利息費用の損金算入制限導入（2017年）**です。これは、多国籍企業が税金の安い国に利益を移転する「ずる賢い税金逃れ」を防ぐための国際的なルール（BEPSプロジェクト）に対応したものです。グループ会社間での高金利な借入金利息について、損金算入額がEBITDA（事業で稼いだ利益）の20%（後に30%に緩和）に制限されるようになりました。

一方で、従業員への優遇策も進みました。2018年には、従業員向けの任意保険料・年金掛金の損金算入上限が、月額100万ドンから**300万ドン**へと大幅に引き上げられたんです。これは、企業が従業員の保障をさらに手厚くできるようになった、ということですね。この時期は、国際的な税務ルールを取り入れつつ、国内企業の健全な発展を促すための「微調整」が行われた時代と言えるでしょう。

### 第4期：新時代の幕開け（2025年～）- 「未来志向の税制へ進化」

そして、2025年10月1日、ベトナム税制は新たな時代を迎えます！新法人所得税法（67/2025/QH15号）は、これまでの法律を全面的に置き換える、まさに「税務の地図が新しくなる」ような大きな変化です。

新法の目的は、これまで政令や通達でバラバラだったルールを**法律レベルで体系的に整理（法典化）**し、税制の透明性と予測可能性を高めることにあります。これにより、企業にとってより分かりやすく、安心して事業ができるようになります。

主な変更点としては、利息費用ルールが法律に明記され、何がOKで何がNGか一目瞭然になったこと。また、環境保護法や広告法など、他の法令に違反する活動から生じた費用は損金にできないことが明確化され、**コンプライアンス遵守**がより重視されるようになりました。さらに、カーボンクレジット取引の優遇や、科学技術研究への寄附が損金算入の対象となる可能性など、**環境対策や社会貢献に積極的な企業が優遇される**ような、未来志向の規定も盛り込まれています。

## まとめ：変化を味方につけて、ベトナムビジネスをもっと楽しもう！

ベトナムの損金不算入制度は、17年間で「厳格な制限」から「経済活動を促進する緩和」、そして「国際基準とコンプライアンスを重視した適正化」へと大きく進化してきました。

「こんなに複雑な税制、私に全部理解できるかな...」と不安に感じる方もいるかもしれませんね。でも大丈夫です！大事なのは、「完璧に理解しなきゃ」と思い込まないこと。基本的な仕組みや流れを理解して、「こういう時は専門家に聞けばいいんだな」という**『勘所』を押さえること**が重要です。

制度は常に変化します。だからこそ、継続的に最新情報をチェックし、少しでも不安を感じたら現地の税務専門家やコンサルタントに相談することをお勧めします。適切な制度理解と専門家との連携により、企業の税務戦略の自由度は格段に向上しています。
