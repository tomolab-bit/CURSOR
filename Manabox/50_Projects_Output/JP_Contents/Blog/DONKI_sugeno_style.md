# ゴチャゴチャしてるのになぜかうまくいく。ドン・キホーテの成功から学ぶ、あなたの仕事をちょっとラクにする「逆張り」の思考法

こんにちは！マナラボの菅野です。

皆さん、ドン・キホーテって行かれますか？あのジャングルのような店内で、目的のものを見つけるのは一苦労...なのに、なぜかまた行きたくなる不思議な魅力がありますよね。そして、ビジネスとしても大成功を収めています。

「なんであのゴチャゴチャしたお店が勝ち続けているんだろう？」って、一度は思ったことありませんか？

実は、その裏には創業者・安田隆夫さんの、常識とはちょっと違う、面白い考え方があるんです。
今日の記事を読むと、私たちが「当たり前」だと思っている仕事のやり方や考え方が、もしかしたら自分を縛り付けているだけかもしれない、って気づけるかもしれません。ちょっと肩の力が抜けるような、新しい視点が見つかるはずですよ。

## 1. 「仮説は必ず間違える」前提で始めよ

まず一つ目。「計画は完璧に！仮説は正しく！」って、ビジネス書にはよく書いてありますよね。でも、安田さんの考えは真逆なんです。『仮説なんて、どうせ間違えるんだから』ってところからスタートする。

え、そんな適当でいいの？って思っちゃいますよね。でも、これってすごく気がラクになる考え方じゃないですか？

最初から100点満点の答えを出そうとすると、怖くて動けなくなってしまいます。でも、「間違えて当たり前」って思えれば、まずは一歩踏出してみようって気になりますよね。

料理で例えるなら、レシピ通りにきっちり作ろうとするんじゃなくて、「まあ味見しながら調整すればいっか」くらいの軽い気持ちで始める感じです。その方が、結果的に自分好みの美味しいものができたりしますから。

ビジネスも同じで、まずやってみて、現実とズレてたら修正する。この繰り返しが、誰にも真似できない、本当に強いものを作り上げる秘訣なんだそうです。なるほど〜って感じですね。

## 2. 思考は7割悲観、行動は7割楽観

二つ目は、これまた面白いんです。『思考は70%悲観、行動は70%楽観』。

この言葉、私、すごく感銘を受けました。というのも、一見するとただの精神論に聞こえるかもしれませんが、実はものすごく科学的で、深い哲学に基づいているからです。安田社長がここまで考えていらっしゃるとは、正直言って意外でした。

安田さんは、この悲観的な思考を「前始末（まえしまつ）」という言葉で表現されています。

これは、何かを始める前に、起こりうる最悪の事態やトラブルを徹底的に洗い出し、先回りして処理しておく、という考え方です。ブレーキのない車でアクセルは踏めないのと同じで、この「前始末」があるからこそ、いざという時に大胆なアクセルを踏み込める、と。

これって、私たちの脳の仕組みにも、すごく合っているんです。

人間の脳には、「ネガティビティバイアス」という性質が備わっています。これは、ポジティブな情報よりもネガティブな情報に注意が向きやすく、記憶に残りやすいという心理的な傾向のことです。太古の昔、危険から身を守って生き延びるために必要だった本能なんですね。

普通にしていると、私たちはこのバイアスのせいで、物事を悪い方向に考えすぎたり、相手の小さなミスがずっと気になったりしてしまいます。

でも安田さんは、この脳の性質を逆手にとって、武器に変えているんです。

つまり、「どうせネガティブなことに目が行くなら、それを徹底的にビジネスの準備（前始末）に活かしてしまえ！」ということなんですね。

この考え方、「失敗は科学、成功はアート」という言葉とすごくリンクするなと感じました。

起こりうる失敗を事前に洗い出す「前始末」は、まさに仮説と検証を繰り返す「科学」のプロセスです。あらゆる失敗のデータを集めて、その確率を極限まで潰していく。そして、その科学的な土台があるからこそ、最後の一歩を「アート」のように、大胆に、直感的に踏み出すことができる。

失敗を恐れるのではなく、科学の実験データとして捉える。この視点、めちゃくちゃ面白いと思いませんか？

## 3. 「苦労して稼いだ金が偉い」という呪いを解く

三つ目は、特に私たち日本人にグサッとくるかもしれません。『苦労して稼いだお金が偉い、は間違い』という考え方です。

「汗水流して頑張ることこそが美しい」みたいな価値観、心のどこかにありませんか？ 私も、ついついそう思っちゃう時があります。これ、心理学でいう「嫌儲バイアス」に近いかもしれません。「お金儲けはなんだか汚いこと」とか「楽して稼ぐのはズルい」みたいに感じてしまう、あの感覚です。

でも安田さんは、「それ、本当に意味ある？」って、バッサリ斬るんです。

時代遅れなやり方で一生懸命頑張っても、結果が出なければ意味がない。ビジネスの目的は価値を生んで成功することであって、苦労することそのものではない、と。

これ、耳が痛いけど、すごく大事なことですよね。頑張ってる自分に酔ってしまうんじゃなくて、「もっとラクに、もっと賢く成果を出す方法はないかな？」って考えること。その方が、ずっと生産的です。

もしあなたが今、頑張りすぎて疲れちゃっているなら、ぜひこの言葉を思い出してみてください。

## 4. 人を育てるな、見つけて「褒めよ」

四つ目は、マネジメントの話です。『人は育てるな、見つけて褒めよ』。

部下や後輩がいる方は、「どうやって育てよう...」って日々悩みますよね。「叱って、教えて、一人前にする」のが当たり前だと思っていませんか？

ドンキ流は違うんです。マネージャーの一番大事な仕事は、部下の良いところを「見つけて、褒める」こと。

ただ、「すごいね！」って漠然と言うだけじゃないのが、この言葉の深いところです。

安田さんの言う「褒める」とは、『その人が誇りに思っているけど、まだ誰も気づいてくれていないことを見つけ出し、具体的に描写すること』なんだそうです。

例えば、「この前の資料、すごく分かりやすかったよ」じゃなくて、「この前の資料の、あのグラフの作り方、すごく工夫されてたね。おかげで会議の論点がすごくクリアになったよ。ああいう視点は君ならではだね」みたいに、具体的に伝える。

本人も意識していなかったかもしれない強みを的確に言葉にしてあげる。そう言われたら、ものすごく嬉しいし、「この人はちゃんと見てくれているんだ」って思いますよね。本当の信頼関係って、こういう細やかなところから生まれるのかもしれません。

## 5. 仕事を「ゲーム化」し、責任感を「与える」

最後の五つ目です。『仕事をゲーム化して、責任感を与える』。

特に若い世代は、「やらされ感」のある仕事って、なかなかモチベーションが上がらないですよね。

安田さんは、仕事を「チームでクリアするゲーム」のように設定するのが、とても上手いんです。

ここで面白いのが、今の若者って「自分が成功したい！」という野心以上に、「チームの仲間に迷惑をかけたくない」という責任感が、強いモチベーションになることを見抜いている点です。

「このゲーム、みんなで力を合わせてクリアしようぜ！」ってなると、不思議と一体感が生まれて、一人ひとりが「よし、僕がこのパートをなんとかしなきゃ」と自発的に動き出す。

これって、責任を「押し付けられる」んじゃなくて、ゲームに勝つために「自分で引き受ける」という感覚ですよね。

「権限委譲」なんて難しい言葉を使わなくても、仕事を面白くするだけで、人は自然とオーナーシップを持って輝き始める。これはすごい発明だと思います。

## ではまとめますね

いかがでしたでしょうか。
ドン・キホーテの創業者、安田さんの仕事術は、

1.  **仮説はどうせ間違える前提で、まず動く**
2.  **準備は最悪を想定し、行動は楽観的に**
3.  **苦労が目的にならないように、賢く成果を出す**
4.  **人を叱るより、誰も気づかない長所を見つけて褒める**
5.  **仕事をゲームにして、みんなで楽しんで責任を持つ**

という、私たちの常識をちょっとひっくり返したようなものばかりでした。

でも、どれも「こうしなきゃいけない」という社会のプレッシャーや思い込みから私たちを解放して、仕事をもっと自由に、もっと面白くするためのヒントに溢れていますよね。

もしあなたが今、仕事の「当たり前」にちょっと窮屈さを感じているなら、この中のどれか一つでも、試してみてはいかがでしょうか。

常識という名のレールから一歩踏み出してみたら、案外、楽しい景色が広がっているかもしれませんよ。
