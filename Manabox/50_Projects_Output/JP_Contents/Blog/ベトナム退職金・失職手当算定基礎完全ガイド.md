# 【ベトナム退職金・完全ガイド】知らないと損する！算定基礎に含まれる手当、含まれない手当

こんにちは！マナラボの菅野です。

ベトナムで会社を経営されている方や人事担当者の方から、こんな相談をよくいただきます。

「従業員が辞めるんだけど、退職金の計算方法が複雑でよくわからない…」
「どの手当を計算に含めて、どれを含めないのか、法律の解釈が難しい！」

そうなんです。ベトナムの退職金制度、特に計算の基礎となる「賃金」の定義は、多くの方が悩むポイントです。

間違った計算をしてしまうと、後々トラブルになりかねません。

**この記事を読めば、ベトナムの退職手当・失職手当の複雑な計算ルールがスッキリ理解でき、自信を持って実務に対応できるようになります。**

少し専門的な話も含まれますが、具体例をたくさん使って、どこよりも分かりやすく解説していきますね。

## なぜ、退職金の計算はこんなに複雑なの？

「昔はもっと単純じゃなかった？」

そう思われる方もいるかもしれません。ベトナムの経済成長とともに、企業の給与体系も多様化しました。基本給だけでなく、さまざまな手当が支給されるようになったのです。

しかし、法律で「どの手当が退職金の計算基礎に含まれるか」が明確に定義されていなかったため、企業と労働者の間で多くのトラブルが発生しました。

「この手当も退職金計算に入れてくれ！」
「いや、それは福利厚生だから対象外だ」

といった具合です。

そこで政府は、労働法や関連政令を改正し、計算の基礎となる賃金の定義をより明確にしました。これが、現在の少し複雑に見えるルールにつながっているのです。

## 「退職手当」と「失職手当」はどう違う？

まず、ベトナムには2種類の退職金制度があることを理解しましょう。

| 種類 | 退職手当 (Trợ cấp thôi việc) | 失職手当 (Trợ cấp mất việc làm) |
| :--- | :--- | :--- |
| **対象者** | 自己都合や契約満了で退職する従業員 | **会社の都合**（組織再編、経営不振など）で職を失う従業員 |
| **計算方法** | 勤続年数1年につき**半月分**の賃金 | 勤続年数1年につき**1ヶ月分**の賃金（最低2ヶ月分保証） |
| **根拠法** | 労働法 第46条 | 労働法 第47条 |

ポイントは、**会社都合の場合は、従業員への補償が手厚くなる**という点です。どちらの手当を支払う必要があるのか、退職理由を明確にすることが重要です。

## 【最重要】結局、何をもとに計算するの？

「計算方法が違うのはわかったけど、その『賃金』って何を指すの？」

ここが一番重要なポイントです。

退職手当も失職手当も、計算の基礎となる賃金は**「退職直前6ヶ月間の労働契約書に記載された平均月給」**と法律（政令145/2020/NĐ-CP第8条）で定められています。

**ポイント：**
*   **直近6ヶ月**の平均
*   **実際に支払われた額ではなく、労働契約書に書かれている金額**がベース
*   ボーナスや残業代は含まない**「固定給」**が対象

つまり、労働契約書に何と書かれているかが、すべてを決めるのです。

## 【判断軸】この手当は算定基礎に含まれる？3つのチェックポイント

「手当の種類が多くて、リストだけだと判断に迷う…」

そんな時のために、ある手当が算定基礎に含まれるかどうかを判断するための、3つのチェックポイントをご紹介します。この考え方を使えば、新しい手当が出てきても応用が効きます。

| チェックポイント | 含まれるケース | 含まれないケース |
| :--- | :--- | :--- |
| **1. 契約書のどこに書いてあるか？** | 賃金（給与）の項目に記載 | 福利厚生の項目に記載 |
| **2. 金額は固定的か？** | 毎月、定額が支払われる | 業績や状況で金額が変動する |
| **3. 支払いの目的は何か？** | 労働そのものへの対価 | 生活支援や見舞金など |

**この3つのチェックポイントのすべてで「含まれるケース」に該当すれば、その手当は算定基礎に含める可能性が非常に高い**と言えます。

特に重要なのが1つ目の「契約書の記載場所」です。ベトナムの法律では、賃金と福利厚生は明確に分けて記載することが求められています（通達10/2020/TT-BLĐTBXH）。

例えば、「住宅手当」について考えてみましょう。

*   **良い例（算定基礎に「含まれる」）：**
    労働契約書の「賃金」セクションに「住宅手当：月額 2,000,000 VND」と**固定額**が明記されている。
*   **悪い例（算定基礎から「除外」される）：**
    契約書の「福利厚生」セクションに記載されている。または、賃貸契約書に応じて実費を支払うなど、金額が毎月変動する。

このように、契約書での扱い方ひとつで、退職金の金額が大きく変わってくるのです。

## あなたの会社の給与はどっち？算定基礎「含む」「含まない」完全リスト

「じゃあ、具体的にどの手当が含まれるの？」

これが一番知りたいところですよね。労働契約に含めるべき賃金・福利厚生の詳細は、労働省の通達（通達10/2020/TT-BLĐTBXH）で具体的に定められています。これを基に、リストアップしました。自社の労働契約書と見比べてみてください。

### ⭕ 算定基礎に「含まれる」賃金・手当

これらは労働契約書の「賃金」の項目に記載されるべきものです。

| 項目 | ベトナム語 | 内容 |
| :--- | :--- | :--- |
| **基本給** | Mức lương | 職務や職階に応じて定められた基本の給与 |
| **職務手当・役職手当** | Phụ cấp lương | 責任の重さや職務の複雑さに応じて支払われる手当 |
| **勤続手当・技能手当**| Khoản bổ sung khác | 勤続年数や特定のスキルに対して固定額で支払われる手当 |
| **その他固定手当** | Khoản bổ sung khác | 住宅手当や通勤手当など、**労働契約書に金額が明記され、毎月固定で支払われる**もの |

### ❌ 算定基礎から「除外される」項目

これらは賃金ではなく「福利厚生」とみなされ、計算基礎には含まれません。労働契約書でも賃金とは別の項目に記載する必要があります。

| 項目 | ベトナム語 | 内容 |
| :--- | :--- | :--- |
| **賞与（ボーナス）** | Thưởng | 業績に応じて支払われる臨時の報酬（労働法104条） |
| **食事手当** | Tiền ăn giữa ca | シフト間の食事補助。福利厚生とみなされる |
| **各種支援金** | Các khoản hỗ trợ | ガソリン代、電話代、育児手当など、生活支援目的の支給 |
| **慶弔見舞金** | Phúc lợi khác | 結婚祝い金、香典、災害見舞金など |
| **残業代・インセンティブ** | - | 実際の労働時間や成果に応じて変動する支払い |


「うちの会社、住宅手当を払ってるけど、これはどっち？」

良い質問ですね。これはまさに、先ほどの「3つのチェックポイント」で判断できます。もし労働契約書の「賃金」の項目に「住宅手当：月額XXXドン」と**固定額が明記**されていれば、それは**算定基礎に含まれます**。一方で、賃金とは別の福利厚生項目として記載されていたり、金額が変動したりする場合は対象外となります。

## 【具体例で理解】実際の計算方法を見てみよう

文章だけだと分かりにくいので、具体的なモデルケースで計算してみましょう。

**モデルケース**
*   勤続年数：5年
*   退職理由：自己都合退職
*   直近6ヶ月の労働契約書上の給与内訳（月額）
    *   基本給：20,000,000 VND
    *   役職手当： 3,000,000 VND
    *   通勤手当（固定額）： 500,000 VND
    *   食事手当（福利厚生）： 1,000,000 VND
    *   業績インセンティブ（変動）：平均 2,000,000 VND

**ステップ１：算定基礎となる月額賃金を計算する**

まず、算定基礎に含まれる項目を合計します。
基本給 (2,000万) + 役職手当 (300万) + 通勤手当 (50万) = **23,500,000 VND**
※食事手当とインセンティブは**除外**します。

この従業員の場合、直近6ヶ月の給与が変動していないため、平均額も23,500,000 VNDとなります。

**ステップ２：退職手当を計算する**

*   勤続年数：5年
*   支給率：1年につき半月分（0.5ヶ月分）
*   計算式：算定基礎賃金 × 支給率 × 勤続年数

**23,500,000 VND × 0.5 × 5年 = 58,750,000 VND**

このケースの退職手当は**58,750,000 VND**となります。

もし、この従業員が**会社都合**で退職（失職手当）する場合、計算式は以下のようになります。

**23,500,000 VND × 1.0 × 5年 = 117,500,000 VND**

金額が大きく変わることがわかりますね。

## これってどうなるの？実務でよくある質問（Q&A）

### Q1: 労働契約書の給与額と、実際の支払額が違う場合はどうなりますか？

**A:** あくまで**労働契約書に記載された金額**が基準となります。

例えば、会社の業績が良く、契約額以上の給与をインセンティブとして支払っていたとしても、その超過分は算定基礎に含まれません。逆に、業績不振で契約額どおりに支払えていなかった場合でも、原則として契約額を基に計算する必要があります。

### Q2: 契約更新を繰り返していて、最後の契約期間が6ヶ月未満の場合は？

**A:** その最後の契約期間における**実際の勤務期間の平均賃金**で計算します。

例えば、最後の契約が3ヶ月で終了した場合、その3ヶ月間の契約上の平均賃金を基に算出します。

### Q3: 労働契約書に「基本給」しか書いておらず、手当は口約束でした。この場合、手当は計算に含められますか？

**A:** 残念ながら、**労働契約書に記載のない手当は算定基礎に含めることができません。**

これが「労働契約書がすべて」と言われる理由です。口約束や、賃金とは別の「福利厚生規定」に記載されているだけの手当は、退職金の計算基礎からは除外されます。従業員とのトラブルを避けるためにも、支給する賃金・手当はすべて労働契約書に正確に記載することが非常に重要です。

## まとめ：正しい知識が、会社と従業員を守ります

今回は、ベトナムの退職金計算という、少し複雑で専門的なテーマを解説しました。

ポイントをもう一度おさらいしましょう。

1.  **2種類の手当**：「退職手当（自己都合）」と「失職手当（会社都合）」があり、計算方法が異なる。
2.  **計算の基礎**：どちらも**「退職直前6ヶ月の労働契約上の平均賃金」**がベース。
3.  **契約書がすべて**：算定基礎に「含まれる」か「含まれない」かは、労働契約書にどう書かれているかで決まる。
4.  **固定的な支払いがカギ**：賞与やインセンティブのような変動給は含まず、毎月決まって支払われるものが対象。

「なんだか難しそう…」と感じたかもしれません。でも、一番大事なのは、**労働契約書を法令に沿って正しく作成し、賃金と福利厚生を明確に分けておくこと**です。

これを徹底するだけで、退職金計算の実務がスムーズになり、不要な労務トラブルを未然に防ぐことができます。

もし、自社の労働契約書や給与規定に不安がある場合は、一度専門家に相談することをお勧めします。正しい知識で備えることが、最終的に会社と、大切な従業員を守ることにつながるのです。
