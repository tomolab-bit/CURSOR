## 海外消費サービスとは？基本的な定義と重要性

**海外消費サービス（Services consumed outside Vietnam）**とは、ベトナム国内で提供されるサービスのうち、実際の消費・利用が海外で行われるサービスを指します。

この概念は、ベトナムの税務実務において極めて重要な位置を占めており、以下の2つの税制に大きな影響を与えます：

### 影響を受ける主要税制
- **VAT（付加価値税）**: 0%税率適用の可否判定
- **コントラクター税**: 課税対象となるか否かの判定

### なぜ重要なのか？

ベトナム進出企業にとって、海外消費サービスの正確な判定は以下の理由で極めて重要です：

1. **税負担の大幅軽減**: VAT 0%税率適用により、実質的な税負担がゼロになる
2. **キャッシュフローの改善**: VAT還付を受けられる可能性
3. **コンプライアンスリスクの回避**: 誤った処理による追徴課税リスクの回避

## 海外消費サービスの判定要件

### 法的根拠の現状

現在のベトナム税法では、「海外消費」の明確な定義が存在しないため、実務上は税務当局の公式見解（公文書）に依拠することになります。

### 判定に必要な2つの要件

税務当局の見解によると、海外消費サービスと認定されるためには以下の**2つの要件**を満たす必要があります：

#### 1. サービス結果の海外市場向け証明
- サービスの完成品・成果物が海外市場向けであること
- 海外での製品消費に関連していること

#### 2. 必要な裏付け書類の準備
以下の書類を適切に準備・保管することが求められます：

| 必要書類 | 具体的な要件 |
|---------|------------|
| **契約書** | 完成したサービスがベトナム国外に移転されることが明記 |
| **海外当事者の確約書** | サービスを海外で消費する旨の正式な確約 |
| **検収報告書** | 海外での検収場所が明確に記載 |
| **国際商慣行に準拠したインボイス** | 輸出取引であることが明確 |
| **銀行送金書類** | 海外からの代金受領を証明 |

## VAT（付加価値税）における税務処理

### 0%税率適用の基本原則

**通達219/2013/TT-BTC第9条第1項**に基づき、以下の条件を満たすサービスには**VAT 0%税率**が適用されます：

```
「輸出サービスには、海外の組織・個人に直接提供され、
ベトナム国外で消費されるサービスが含まれる」
```

### 0%税率適用の具体的メリット

1. **売上VAT**: 0%で計算（実質的に課税なし）
2. **仕入VAT**: 通常通り控除・還付可能  
3. **実質的な税負担**: ゼロまたはマイナス（還付）

### 実際の適用事例

**【事例1】グラフィックデザインサービス**
```
公文書3705/TCT-CS の事例：
- ベトナム企業が日本企業向けにPhotoshopでグラフィックデザイン作成
- PSDファイル形式で電子的に納品
- 日本国内で利用される
→ VAT 0%税率適用が認められた
```

**【事例2】ソフトウェア開発サービス**  
```
公文書12492/CT-TTHTの事例：
- ベトナム企業が海外企業向けにシステム開発
- 完成品は海外サーバーに納品・運用
- 海外ユーザーのみが利用
→ VAT 0%税率適用
```

## コントラクター税の適用関係

### 基本的な課税原則

**通達103/2014/TT-BTC**に基づき、コントラクター税の課税対象は以下のように規定されています：

#### 課税対象となるケース
- ベトナム国内でサービスを提供し、**ベトナム国内で消費**される場合
- ベトナム国外でサービスを提供し、**ベトナム国内で消費**される場合

#### 課税対象外となるケース
- **海外で提供され、海外で消費される**サービス

### 実務上の重要なポイント

**【注意】** 一部の税務当局では、「海外消費であってもベトナム国内で実行されるサービス」についてはVAT 10%を適用するという見解も存在します。

```
公文書2002/TCT-CSの事例：
デンマーク企業向けのカスタマーサービス、債権管理、IT サービスについて、
「海外顧客向けだが、ベトナム国内で実行・消費される」として
VAT 10%が適用された事例
```

## 実務で注意すべき適用除外項目

以下のサービスは、海外消費であっても**VAT 0%税率の適用外**となります：

### 金融・保険関連サービス
- 海外向け保険再保険
- 技術移転・知的財産権の海外移転
- 海外への資本移転・信用供与・証券投資
- 金融デリバティブサービス

### 通信・郵便サービス  
- 海外向け郵便・通信サービス（非関税地域向けを含む）
- プリペイドカードの海外販売

### 資源・嗜好品
- 未加工の天然資源・鉱物の輸出
- 輸入後再輸出するタバコ・酒類・ビール

### ベトナム国内で提供される各種サービス
- スポーツ競技・芸術公演・文化娯楽
- 会議・ホテル・研修・広告・旅行業
- オンライン決済サービス
- 製品販売・流通に付随するサービス

## 具体的な事例と税務当局見解

### 【実践事例】研究開発サービスの複雑なケース

**設定**: 
- A社：ベトナムの外資系企業
- B社：日本の研究開発会社
- 契約内容：A社がB社の研究開発サービスを利用し、必要な物品・サービスもA社が調達・支払い

**税務判断のポイント**:

```
1. 研究活動の実施場所：日本国外（ベトナム国外）
   → B社はベトナムのコントラクター税対象外

2. 研究成果の消費場所：ベトナム国内
   → B社はベトナムのコントラクター税対象

3. 適用される制度：
   - B社が通達103/2014/TT-BTC第8条の条件を満たさない場合
   → A社（ベトナム企業）がB社に代わって納税義務を負う
```

## 実務上の留意点とリスク対策

### 1. 書類準備の徹底

海外消費サービスの適用を受けるためには、判定要件で示した書類を**事前に**適切に準備することが不可欠です。

**推奨される実務フロー**:
```
契約締結時 → 海外消費条項の明記
↓
サービス提供中 → 進捗管理と証憑収集  
↓
完了時 → 海外検収書の取得
↓
請求・回収時 → 海外送金書類の保管
```

### 2. 税務当局との事前相談

複雑なケースや新しいビジネスモデルについては、税務申告前に**所轄税務署への事前相談**を強く推奨します。

### 3. 継続的な法令改正チェック

ベトナムの税制は頻繁に改正されるため、最新の通達や公文書を定期的にチェックし、自社の取引への影響を評価することが重要です。

### 4. 専門家との連携体制構築
