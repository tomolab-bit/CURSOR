# ベトナムAIC社汚職事件完全ガイド：グエン・ティ・タイン・ニャン会長の17年間の不正行為

こんにちは！マナラボの菅野です。

最近、ベトナムで事業をしている日系企業の経営者から、こんな相談をいただきました。

「ベトナムのAIC社という会社の汚職事件が話題になっているようですが、これは私たちの事業に何か影響がありますか？ベトナムの入札制度って、本当に大丈夫でしょうか...」

実は、この相談、めちゃくちゃ多いんです。

2022年以降、ベトナムで発覚したAIC社（国際進歩株式会社）の大規模汚職事件は、同国の公共調達制度に大きな衝撃を与えました。会長のグエン・ティ・タイン・ニャン被告は17年間にわたって組織的な入札談合を行い、国家に約4,828億ドン（約29億円）もの損害を与えたとされています。

**この記事を読むと、AIC社汚職事件の全体像がスッキリ理解できて、ベトナムでの事業運営で注意すべきポイントがバッチリわかります。**

正直、汚職事件の話なので少し重い内容もありますが、なるべくわかりやすく、実際のビジネスで役立つように説明していきますね。

## AIC社ってどんな会社だったの？

「AIC社って聞いたことないけど、どんな会社だったの？」

こんな声をよく聞くようになったんです。実は、AIC社はベトナムではかなり有名な企業だったんですね。

### ベトナム最大500社に入る大企業

AIC社（正式名称：国際進歩株式会社）は、2005年に36歳のグエン・ティ・タイン・ニャンが会長に就任してから、急成長を遂げた企業でした。

AIC社は医療機器・設備、教育関連設備、不動産開発、エネルギー事業、科学技術分野など、多岐にわたる事業を展開していました。企業規模も大きく、29の子会社を擁し、数千人の社員を抱えるベトナム最大500社にランクインする大企業でした。

「これってすごい会社じゃない？」

そうなんです。表面的には、AIC社はベトナムの成功企業の典型例に見えました。しかし、その成長の裏には、極めて組織的で巧妙な汚職システムが隠されていたんです。

## なぜ17年間もバレなかったの？

「こんな大規模な汚職が17年間も続いたなんて信じられない...」

そう思うのも当然です。実は、ニャン被告は入札談合を「企業の戦略」として体系化していたんですね。

### 70ステップの入札談合マニュアル

AIC社では、なんと「70ステップの入札談合マニュアル」という内部文書が作成されていました。これは、入札関連法規を形骸化させるあらゆる手口が網羅された、まさに「汚職の教科書」だったんです。

このマニュアルには、例えばステップ19では「希望どおりの結果が出るよう、価格査定業者と通じて価格査定を行う」こと、ステップ26では「案件ごとにダミー企業（いわゆる"青軍""赤軍"）を用意し、当社が確実に落札できるよう入札を取り計らう」ことなどが明記されていました。

### 青軍・赤軍方式とは？

これは、AIC社が開発した極めて巧妙な談合システムです※。

※注：青軍・赤軍方式は入札談合でよく使われる手法の一つですが、AIC社はこれを組織的に体系化し、70ステップのマニュアルに組み込んで大規模に実行した点が特徴的です。

このシステムの仕組みは次のようなものでした。まず、AIC社が複数のダミー会社を設立し、表面上は競争入札を装います。しかし実際は全てAIC社のコントロール下にあり、発注機関と事前に結託して仕様書を有利に設定していました。

「これって完全に犯罪じゃない？」

その通りです。しかし、このシステムが17年間も機能し続けた理由があります。

## 癒着の仕組み：内部委員会の設置

ニャン被告は、汚職を継続するために社内に「内部委員会（Ban nội bộ）」を設置していました。

### 内部委員会の役割

内部委員会は、談合で得た不正利益を現金化し、正式な帳簿に載らない形で資金を管理していました。そして、地方幹部への「謝礼」や「潤滑油」として、党委書記・人民委員長などに袖の下を渡す役割を担っていたのです。

「これって組織ぐるみの犯罪じゃない？」

まさにその通りです。AIC社は企業として、汚職を「事業戦略」として位置づけていたんですね。

### 2009年：ドンナイ省での「長期戦略」の始まり

2009年、ドンナイ省で新総合病院の建設が始まりました。投資総額約2兆ドン（約120億円）という大型プロジェクトです。

AIC社は、この案件で医療機器16件の調達を狙いました。しかし、単なる入札参加ではありませんでした。ニャン被告は、この案件を「長期戦略」として位置づけ、12年間にわたって継続的に賄賂を供与し続けたのです。

**結果：**
- 契約総額：約6,650億ドン（約39.9億円）
- 水増し分：約1,520億ドン（約9.1億円）（国家損害額）
- 賄賂総額：約438億ドン（約2.6億円）

ドンナイ省の省委書記長、人民委員長、病院院長らが次々と汚職に染まっていきました。12年間という長期間、誰も気づかなかったのは、AIC社の「内部委員会」が巧妙に資金を管理していたからです。

### 2013年：バクニン省での「競争」の実態

2013年、バクニン省で6つの郡病院を新築するプロジェクトが始まりました。AIC社は医療機器調達6件のうち3件を落札しました。

しかし、この事件には興味深い特徴がありました。AIC社の賄賂総額（約279億ドン・約1.7億円）が、国家損害額（約260億ドン・約1.6億円）を上回ったのです。これは、AIC社と同時に別の企業も賄賂を供与していたためでした。

つまり、同じ案件で複数の企業が「競争」していたのは、入札の落札ではなく、賄賂の額だったということです。

### 2013年：クアンニン省での「完璧な犯罪」

同じ2013年、クアンニン省の産科・小児科病院プロジェクトでもAIC社が活躍しました。契約額約2,300億ドン（約13.8億円）で落札し、価格を約22%吊り上げました。

国家損害額は約510億ドン（約3.1億円）に達しましたが、この事件では公務員への収賄罪の立件がありませんでした。つまり、談合による不当利得の全てがAIC社側の利益となっていたのです。

これは、AIC社の「70ステップマニュアル」が完璧に機能した例でした。賄賂の痕跡を残さずに、巨額の利益を確保していたのです。

### 2014年：ホーチミン市での「家族の事情」

2014年、ホーチミン市のバイオテクノロジーセンターで12の実験室設備調達8件の入札が行われました。AIC社は全件を落札し、市場価格より40%も高い価格で契約を結びました。

国家損害額は約946億ドン（約5.7億円）に達しました。この事件で注目すべきは、センター所長のズオン・ホア・ソー被告の証言でした。

彼は、AIC社から受け取った賄賂約144億ドン（約0.9億円）の一部を子息の留学費用に充てていたと供述したのです。汚職の動機が「家族の将来」だったという、皮肉な現実が明らかになりました。

### 2016年：VNCERTでの「圧力の連鎖」

2016年、情報通信省管轄のVNCERT（国家緊急コンピュータ救援センター）で、全国的なネットワーク監視・分析システム機器調達プロジェクトが始まりました。総予算950億ドン（約5.7億円、うち機器費706億ドン・約4.2億円）の大型案件です。

AIC社は入札資格がないにもかかわらず、書類を偽装して参加しました。そして談合により約706億ドン（約4.2億円）で落札したのです。実際の市場価格は約530億ドン（約3.2億円）程度だったため、国家損害額は約172億ドン（約1.0億円）に達しました。

VNCERT所長のグエン・チョン・ズオン被告は、法廷で衝撃的な証言をしました。「当時、上司である大臣からAIC社を落札させるよう大きな圧力を受けていた」と。

これは、汚職が単なる企業と公務員の癒着ではなく、組織的な「圧力の連鎖」によって成り立っていたことを示していました。

### 2012～2017年：ホーチミン市での「日常化した贈り物」

2012年から2017年にかけて、ホーチミン市の教育・医療・環境部門で計230件の入札不正が行われました。AIC社は教育部門だけで134件の入札を落札し、約1,420億ドン（約8.5億円）の国家損害を生み出しました。

この事件の特徴は、汚職が「日常化」していたことでした。入札実施前にAIC社側が所管部署の幹部を訪問し「贈り物」を手渡すという慣行が、もはや当たり前の行為となっていたのです。

教育部門だけで約1,420億ドン（約8.5億円）、関連分野全体（教育・医療・環境）では約1,450億ドン（約8.7億円）もの公費が、子どもたちの教育環境改善ではなく、汚職に使われていました。

## 国家への深刻な影響

AIC社の汚職事件は、ベトナム国家に深刻な影響を与えました。

### 財政的損失の具体的な意味

**総損害額：** 約4,828億ドン以上（約29億円）

**4,828億ドン（約29億円）の規模感：**
- ベトナムの地方都市の年間予算に相当
- ベトナム国民1人あたり約4,828ドン（約29円）の税金が無駄に
- 日本の中小企業の年間売上高に匹敵

この「財政的損失」とは、単純に「お金が無駄になった」という意味ではありません。なぜ国家の損失になったのか、その仕組みを詳しく説明します。

### なぜ国家の損失になるのか？

**基本的な仕組み：**
公共調達では、本来であれば市場価格で適正に調達されるべきです。しかし、AIC社の談合により、実際の市場価格よりも高い価格で契約が結ばれました。この「価格の差額」が、そのまま国家（つまり国民の税金）の損失となるのです。

**具体例で説明：**
例えば、医療機器の市場価格が500億ドン（約3億円）だったとします。しかし、AIC社の談合により、同じ機器を650億ドン（約3.9億円）で購入することになりました。この差額150億ドン（約0.9億円）が、そのまま国家の損失となります。

**なぜ「損失」なのか：**
- 本来500億ドンで済むはずの調達に、150億ドン多く支払った
- この150億ドンは、他の公共サービスに使えるはずだった税金
- 結果として、国民全体が150億ドン分のサービスを受けられなくなった

**損失の種類：**

1. **直接的な損失**：価格の水増し分そのもの
2. **機会損失**：本来できたはずの他の投資ができなくなった
3. **社会的損失**：公共サービスの質の低下
4. **経済的損失**：健全な競争の阻害による市場の歪み

**具体例：**
ドンナイ省病院事件では、1,520億ドン（約9.1億円）の損失により、他の病院の設備購入ができなくなり、医療従事者の研修費が削減され、患者の治療費補助も減らされてしまいました。つまり、1,520億ドン分の医療サービスが失われたのです。

具体的には以下のような深刻な影響を意味しています。

#### 1. 国家予算の直接的な無駄遣い

通常の入札では市場価格で適正に調達されます。例えば、医療機器が500億ドン（約3億円）で調達可能な場合、AIC社の談合入札では価格を水増しして同じ医療機器を650億ドン（約3.9億円）で調達します。この差額150億ドン（約0.9億円）が「財政的損失」となるのです。

#### 2. 具体的な損失の内訳

**ドンナイ省病院事件の例：**
```
本来の適正価格：約5,130億ドン（約30.8億円）
AIC社の落札価格：約6,650億ドン（約39.9億円）
財政的損失：約1,520億ドン（約9.1億円）
```

この1,520億ドン（約9.1億円）の損失により、他の病院の設備購入ができなくなり、医療従事者の研修費が削減され、患者の治療費補助も減らされてしまいました。

**バクニン省6病院事件の例：**
```
本来の適正価格：約740億ドン（約4.4億円）
AIC社の落札価格：約1,000億ドン（約6億円）
財政的損失：約260億ドン（約1.6億円）
```

この260億ドン（約1.6億円）の損失により、6つの病院の運営費が圧迫され、医薬品の購入予算が削減され、医療スタッフの給与改善もできなくなってしまいました。

#### 3. 間接的な損失と機会損失

本来であれば、新しい医療機器の導入、病院の建物改修、医療従事者の研修プログラム、患者向けサービスの向上などに投資できたはずでした。しかし、汚職によりこれらの機会が失われ、長期的には医療サービスの質の低下、患者の治療機会の減少、医療従事者の士気低下、地域医療の後退といった深刻な社会的影響をもたらしたのです。

#### 4. 税金の無駄遣い

ベトナム国民の税金は適正価格の1.3倍～1.4倍で支払われ、本来の公共サービスに使えず、汚職に関与した人たちの懐に入ってしまいました。具体的には、1,520億ドン（約9.1億円）の損失はベトナム国民1人あたり約1,520ドン（約9.1円）の税金が無駄になったことを意味し、4,828億ドン（約29億円）の総損失は国民1人あたり約4,828ドン（約29円）の税金が無駄になったことを意味します。

#### 5. 経済への波及効果

汚職は他の企業にも深刻な影響を与えました。優秀な企業が入札から排除され、技術革新の機会が失われ、市場の健全な発展が阻害されました。また、外国企業の信頼失墜により、ベトナムのビジネス環境への懸念が高まり、投資の減少や経済成長の鈍化といった問題も生じています。

これらの汚職事件により、医療・教育の充実に使えるべき予算が浪費され、公共サービスの質的向上の機会が失われ、不適切なベンダーによるシステム品質の低下といった深刻な問題が生じました。

### 社会的信頼への打撃

汚職事件は行政への信頼失墜をもたらしました。公行政の公正さへの疑問が生じ、市民の政府への信頼が大幅に低下し、汚職防止制度の不備が露呈してしまいました。

**裁判での厳しい指摘：**
「本件は企業と権限者がもたれ合う利益集団の腐敗の典型例」

## まとめ：健全な競争環境の構築に向けて

AIC社汚職事件は、ベトナムの公共調達制度に大きな傷跡を残しました。しかし、この事件は同時に、健全な競争環境を構築するための重要な教訓でもあります。

### 事件から学ぶべきこと

**1. 組織的汚職の危険性**
- 企業ぐるみの不正は必ず露見する
- 短期的な利益よりも長期的な信頼が重要
- コンプライアンス体制の重要性

**2. 行政の透明性の重要性**
- 入札プロセスの透明性確保
- 監視体制の強化
- 市民の信頼回復

**3. 国際的な基準への対応**
- グローバルなコンプライアンス基準
- 透明性の高いガバナンス
- 持続可能な事業運営

### 日系企業へのメッセージ

ベトナムでの事業運営においては、以下の点を心がけることが重要です：

**適正な競争：**
- 公正な入札参加
- 透明性の高い事業運営
- 長期的な信頼関係の構築

**リスク管理：**
- 事前の十分な調査
- 適切なコンプライアンス体制
- 継続的な監視・改善

**社会的責任：**
- 現地社会への貢献
- 持続可能な事業発展
- 倫理的な経営の実践

AIC社事件は確かに深刻な問題でしたが、これを機にベトナムの公共調達制度はより透明で公正なものへと改善されつつあります。日系企業にとっては、適正な競争を通じて信頼を築き、長期的な成功を目指すチャンスでもあるのです。
