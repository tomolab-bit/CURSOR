こんにちは！マナラボの菅野です。

今日は、2026年1月1日に施行される新会計通達「99/2025/TT-BTC」の中で、**製造業や卸売業の方々にとって、おそらく最も影響が大きい変更**についてお話ししたいと思います。

それは、勘定科目`611（仕入）`と`631（製造原価）`の廃止です。

「勘定科目が1つ2つなくなるくらい、大したことないんじゃない？」と思われるかもしれません。

でも、実はこれ、単なる勘定科目の削除ではないんです。ベトナムの在庫評価や原価計算の会計実務が、根本的に変わってしまう大きな変更なんです。

もしこの変更を理解せずに2026年を迎えてしまうと、月次決算や年次決算が正しく行えなくなってしまう可能性もあります。

今日は、この2つの勘定科目がなぜ廃止されるのか、そして今後どんな会計処理が求められるのか、具体的な仕訳例も交えながら、できるだけわかりやすくお伝えしていきますね。

この記事を読んでいただければ、新通達への対応で何をすべきかが明確になります。

## 1. 勘定科目611（仕入）がなくなると、どうなるの？

### まず、旧制度での`611`の役割を理解しましょう

旧通達200号では、企業は在庫管理方法として、実は2つの方法から選べました。

1つ目が**棚卸計算法**です。これは、期中は仕入を費用勘定である`611（仕入）`に計上して、期末に実地棚卸（実際に在庫を数えること）を行って、在庫分を資産勘定（`152`や`156`など）に振り替える方法です。

2つ目が**継続記録法**です。これは、仕入の都度、在庫を資産勘定（`152`や`156`など）に直接計上して、販売時に売上原価を計上する方法です。

`611`は、このうち「棚卸計算法」を採用している企業が使っていた、期中の一時的な費用勘定でした。簡単に言えば、**中間的な集計場所**みたいなものですね。

この勘定は期中に仕入を集計するんですが、**期末には必ず残高がゼロになる**という特性がありました。

### なぜ廃止されたのか？

新通達で`611`が廃止された理由は、**中間勘定を経由せず、より直接的な会計処理を実現するため**です。

ここで重要なポイントを整理しておきますね。

新通達では、`611`勘定の使用が明確に禁止されています。

ただし、**棚卸計算法そのものは新通達でも継続されています**。ここ、勘違いしやすいポイントなんです。

私の個人的な推測になりますが、財務省の判断としては、`611`という中間勘定を使わなくても、棚卸計算法は実現できると考えたのではないかと思います。

あくまでも私の意見ですが、つまり、期中は直接資産勘定（`152`や`156`など）に記録して、期末に実地棚卸で調整する方法に統一されたということだと理解しています。

### 仕訳はどう変わるの？具体例で見てみましょう

実際に仕訳がどう変わるのか、具体例で見てみましょう。

#### 【旧制度】棚卸計算法で`611`を使っていた場合

まず、仕入時にはこうなっていました。

```
借方）仕入 (TK 611)             100
借方）仮払付加価値税 (TK 133)      10
    貸方）買掛金 (TK 331)           110
```

そして、期末決算時に実地棚卸の結果、在庫が70、売上が30だったとします。するとこうなります。

```
借方）商品 (TK 156)             70 （在庫分）
借方）売上原価 (TK 632)         30 （売上分）
    貸方）仕入 (TK 611)           100
```

こうして、期末には`611`の残高がゼロになるんです。

#### 【新制度】`611`を使わない方法

新制度では、仕入時に直接資産として計上します。

```
借方）商品 (TK 156)             100
借方）仮払付加価値税 (TK 133)      10
    貸方）買掛金 (TK 331)           110
```

そして、期末決算時に実地棚卸で確定した売上分を費用へ振り替えます。

```
借方）売上原価 (TK 632)         30 （実地棚卸で確定した売上分）
    貸方）商品 (TK 156)           30
```

中間勘定を使わず、より直接的な処理ができるようになりました。

シンプルに言えば、**余計な経由地を通らずに、直接目的地に届けるようなイメージ**ですね。

## 2. 勘定科目631（製造原価）がなくなると、どうなるの？

### `631`も、実は中間勘定だった

勘定科目`631（製造原価）`は、旧通達200号で棚卸計算法を採用している企業が使っていた勘定でした。

具体的には、期中に発生した製造費用（`621`：直接材料費、`622`：直接労務費、`627`：製造間接費）を期末に一度`631`に集計して、そこから`154（仕掛品）`勘定に振り替えるという流れでした。

これも`611`と同様に、**期末には残高がゼロになる中間勘定**です。

### なぜ廃止されたのか？

`631`が廃止された理由は、`611`と同様に、中間勘定を経由せずに原価計算を実現するためです。

私の個人的な推測になりますが、財務省は、`631`という中間勘定を使わなくても、製造費用を`154（仕掛品）`に直接振り替えることで、棚卸計算法による原価計算は実現できると判断したのではないかと思います。

あくまでも私の意見ですが、こう解釈しています。

### 仕訳はどう変わるの？

期末に製造費用を仕掛品に振り替える際の仕訳が変わります。

#### 【旧制度】`631`を使っていた場合

まず、期末に各製造費用を`631`へ集計します。

```
借方）製造原価 (TK 631)           150
    貸方）直接材料費 (TK 621)         80
    貸方）直接労務費 (TK 622)         50
    貸方）製造間接費 (TK 627)         20
```

そして、`631`から`154`へ振り替えます。これで`631`の残高はゼロになります。

```
借方）仕掛品 (TK 154)             150
    貸方）製造原価 (TK 631)           150
```

#### 【新制度】`631`を使わない方法

新制度では、各製造費用を直接`154`へ振り替えます。

```
借方）仕掛品 (TK 154)             150
    貸方）直接材料費 (TK 621)         80
    貸方）直接労務費 (TK 622)         50
    貸方）製造間接費 (TK 627)         20
```

中間勘定`631`を経由せず、直接的な処理が可能になりました。

これも、余計な手間を省いて、よりシンプルになったということですね。

## 3. 実際に何を準備すればいいの？

### 2025年度末に何をすべきか？

新通達の第29条では、旧通達の勘定科目の残高を、新通達の性質が類似した勘定科目に移行することが求められています。

ただし、`611`と`631`は期中に発生して、期末決算で残高がゼロになる費用勘定・中間勘定です。

だから、移行作業の本質は**「2025年度の期末決算を旧通達に基づいて正しく完了させ、これらの勘定の残高がゼロであることを確認すること」**になります。

2026年1月1日以降は、これらの勘定科目を**使用しない新しい業務プロセス**を構築する必要があります。

### 準備チェックリスト

影響を受ける可能性のあるすべての企業、特に製造業・卸売業の方は、以下の準備を早期に開始することをお勧めします。

まずは、**プロセスの確認と定義**から。

- 自社が現在「棚卸計算法（`611`使用）」を採用しているか確認してください。
- 2026年1月1日以降は、`611`や`631`という中間勘定を使わずに処理する新しい仕訳プロセスを定義して、社内の会計マニュアルを更新する必要があります。
- 棚卸計算法を継続する場合でも、中間勘定を使わない方法への移行が必要です。

次に、**会計システムの対応**です。

- 使用している会計ソフトウェアのベンダーに、新通達99号への対応状況（特に`611`、`631`の廃止）を確認してください。
- 必要に応じて、ソフトウェアのバージョンアップや設定変更の計画を立てましょう。例えば、仕入伝票の計上先勘定の変更や、原価計算モジュールのロジック変更などです。

最後に、**従業員への周知と研修**です。

- 経理・会計担当者に対して、変更の背景と新しい業務プロセスについての研修を実施してください。

## まとめ

勘定科目`611`と`631`の廃止は、単なる勘定体系の変更ではありません。

これは、**中間勘定を経由せず、より直接的な会計処理を実現する**という、ベトナム財務省の方針転換なんです。

重要なポイントをまとめますね。

**棚卸計算法は新通達でも継続されています**。ただし、`611`や`631`という中間勘定を使わない方法で実現する必要があります。

`611`や`631`は期中に集計されて、期末には残高がゼロになる中間勘定でした。

私の個人的な推測ですが、財務省は、これらの勘定を使わなくても棚卸計算法は実現できると判断したのではないかと思います。あくまでも私の意見としてお伝えしています。

これまで「棚卸計算法（`611`使用）」を採用してきた企業は、**中間勘定を使わない新しい会計処理方法への移行**が必要になります。
