# 【ベトナム貿易の罠】たった1つのコードミスが会社を潰す。決定1357/QD-TCHQ、生き残るための唯一の教科書

**「この取引、どの通関コードを使えばいいんだろう…？」**

あなたの部下や通関業者が日々行っている、その何気ない「コード選択」。しかし、そのたった一つの判断ミスが、数ヶ月後、あなたの会社に**数千万円規模の追徴課税**という形で牙を剥くとしたら…？ 特に、**輸出加工企業（EPE）**として特別な恩恵を受けるあなたの会社は、税関の最重要監視ターゲットであることを、まず認識してください。

これは脅しではありません。ベトナムでビジネスを行う以上、避けては通れない現実です。

ベトナム税関総局が定めた**「決定1357/QD-TCHQ」**。これは単なるルールブックなどという生易しいものではありません。あなたの会社の生殺与奪を握る、絶対的な「法律」です。

この記事では、ベトナム通関の修羅場をくぐり抜けてきた専門家として、この決定1357/QD-TCHQを骨の髄まで分解し、あなたの会社を破滅から救うための「生存戦略」を授けます。一言一句、読み飛ばさないでください。あなたの会社の未来がかかっています。

## 1. 決定1357/QD-TCHQとは何か？ - あなたが従うべき唯一のルール

結論から言います。**決定1357/QD-TCHQは、ベトナム税関との戦いにおいて、あなたが唯一手にできる「地図」であり「武器」です。**

| 項目 | **あなたのビジネスにとっての意味** |
| :--- | :--- |
| **発行機関** | ベトナム税関総局 (General Department of Customs) - **あなたの会社を調査する組織そのもの** |
| **文書番号** | 1357/QD-TCHQ |
| **効力発生** | 2021年6月1日から - **今、この瞬間も有効なルール** |
| **権威性** | **交渉の余地なき絶対的な法律。**「知らなかった」は通用しません。 |

この決定によって、ベトナム全土のルールが統一されました。つまり、あなたの会社は、ベトナムのどこにいても、このルールに基づいて評価され、そして裁かれます。特に**EPE**は、その優遇措置の大きさから、**最も厳格な適用と監視の対象**となります。

<a name="epe-sxxk-diff"></a>
## 2. まず敵を知れ：EPEとSXXK、その決定的な違い

この法律を理解する上で、まずEPE（輸出加工企業）とSXXK（輸出製品製造）という2つのプレイヤーの違いを明確に区別する必要があります。どちらも「輸出のために原材料を免税で輸入する」という点で共通していますが、その本質とリスクは全く異なります。

| 項目 | **SXXK (輸出製品製造)** | **EPE (輸出加工企業)** |
| :--- | :--- | :--- |
| **本質** | 事業形態・税関上の**管理類型**。一般企業が行う活動。 | 企業形態そのもの。**会社のライセンス**の種類。 |
| **場所** | 国内どこでも可。 | 輸出加工区（EPZ）や工業団地（IZ）内など**地理的に隔離**されている必要がある。 |
| **税関上の位置づけ** | 国内企業。 | **「外国」と見なされる**非関税地域。 |
| **最大のリスク** | **決算報告（精算報告）**で、輸入した原材料がすべて輸出されたことを証明できない場合、遡って課税される。 | 企業そのものが「治外法権」的な存在のため、**国内への不正流出**を常に疑われる。在庫差異は即、密輸の嫌疑に繋がる。 |

### 法的根拠：なぜEPEとSXXKは全く違う存在なのか

この違いは、単なる運用上の区分ではありません。ベトナムの法律で明確に定められた、**法的に全く異なる存在**なのです。

**EPEの法的根拠：**
- **関税法（Law No. 107/2016/QH13）第4条第2項**が、EPEを「**非関税地域（khu phi thuế quan）**」として規定。つまり、法律上EPEは「外国」として扱われます。
- **付加価値税政令（Decree 181/2025/ND-CP）第3条**で、EPEは特別な事業体として明記されています。
- EPEは**投資ライセンスの種類そのもの**であり、設立時点から特別な法的地位を持ちます。

**SXXKの法的根拠：**
- **通達38/2015/TT-BTC（通達39/2018で改正）**および**政令134/2016/ND-CP（政令18/2021で改正）**で、「輸出製品製造のための輸入原材料」の管理方法として規定。
- SXXKは企業形態ではなく、**一般企業が行う活動の一種**として扱われます。
- 場所の制限はなく、決算報告で原材料の使用を証明する義務があります。

**決定的な違いを示す条文：**

```
関税法第4条第2項：
「外国から非関税地域（khu phi thuế quan）に輸入され、
非関税地域内でのみ使用される貨物は課税対象外」
```

この条文が意味するのは、**EPEへの輸入＝外国への輸出**ということです。つまり、EPEの倉庫は法律上「ベトナム国外」なのです。

だからこそ、EPEが原材料を倉庫から持ち出して国内で使用・販売する場合、それは「外国からベトナムへの輸入」となり、**用途変更手続き（A42コード）**が絶対に必要になるのです。

---

あなたの会社がEPEであるならば、あなたは常に「国境」を管理しているのと同じ責任を負っています。SXXKは活動ベースの管理ですが、EPEは存在そのものが税関の監視対象なのです。この認識の違いが、あなたの会社を守れるかどうかの分岐点となります。

<a name="export-codes"></a>
## 3. 【輸出編】利益を守るためのコード戦略

### <a name="b11"></a>B11：事業輸出（Export business）
全ての基本となるコード。しかし、「基本」ほど怖いものはありません。
*   **用途**: 販売契約に基づく、最も単純な製品輸出。
*   **経営者の視点**: あなたの会社の取引が、後述する特殊なケース（B13やEシリーズ）に**絶対に該当しない**と確信できる場合のみ、このコードは安全です。安易な選択は命取りです。

### <a name="b13"></a>B13：輸入済貨物の輸出（Export of imported goods）
**ここが地雷原です。** 多くの企業がここで間違いを犯し、莫大な追徴課税を受けています。

*   **鉄則**: このコードが許されるのは、輸入した貨物に**指一本触れず（一切加工せず）**、そのままの状態で海外へ送り返す場合**のみ**です。
*   **具体的な罠**:
    *   「ちょっと検品のために開封した」「簡単な組み立てだけした」→ **それはもはや「未加工」ではありません。即アウトです。**
    *   特に**EPE企業**が、輸入したものの使わなくなった原材料を処分する際、このコードの判断を誤れば、過去の免税メリットがすべて剥奪され、破滅的な追徴課税が待っています。
*   **税関の視点**: 税関は「本当に未加工か？」を悪魔の目で見ています。製造日報、在庫元帳、すべてが証拠となります。少しでも疑いがあれば、彼らは容赦しません。

### <a name="e-series"></a>E42 / E52 / E62：EPE・加工・SXXK - あなたは正しく使い分けているか？
EPE、委託加工、SXXK。これらの優遇制度を享受しているなら、コードの使い分けは絶対です。

*   **E42 (EPE製品)**: EPEライセンスを持つあなたの会社が「製造した製品」を輸出するコード。**EPEであるというだけで、あなたの申告は常に特別な監視下にある**と心得てください。
*   **E52 (委託加工品)**: 海外から原材料を「預かって加工した製品」を輸出するコード。
*   **E62 (SXXK品)**: 免税で「輸入した原材料で製造した製品」を輸出するコード。
*   **致命的な間違い**: **加工せず輸出するB13**と、**加工して輸出するE62**。この2つを混同することが、どれほど危険か理解していますか？ 税務調査官は、この違いを突破口に、あなたの会社の管理体制の欠陥を徹底的に暴きにきます。

<a name="import-codes"></a>
## 4. 【輸入編】コストとリスクを支配するコード戦略

### <a name="a11-a12"></a>A11 / A12：事業輸入と生産輸入 - 課税の分水嶺
*   **A11 (消費・事業輸入)**: 国内販売・消費目的。原則、**関税・VATが課税**されます。
*   **A12 (生産輸入)**: 国内生産目的。投資優遇などにより**減免税の可能性**があります。
*   **経営者の視点**: この選択は、キャッシュフローに直結します。A12を適用するには「生産のため」という明確な証拠と計画が必須です。安易な節税目的での使用は、後の調査で必ずバレます。

### <a name="a42"></a>A42：用途変更 - 脱税を疑われないための絶対的防衛策
**あなたの会社のコンプライアンス意識が試される、最重要コードです。**

*   **絶対的義務**: **特にEPE**が、免税で輸入した物品を、輸出以外の目的（国内販売、廃棄、消費）に使う場合、このコードで**自ら税金を納める**必要があります。これは、自社の「外国領土」からベトナム「国内」へ商品を正式に輸入する、極めて重要な手続きです。
*   **破滅への道**: この申告を怠ることは**「脱税」と「密輸」そのもの**です。税関に見つかった場合、追徴課税はもちろん、延滞税、高額な罰金が科され、会社の信用は地に落ちます。不良在庫の国内処分も、この手続きなくしてはあり得ません。

### <a name="e-import"></a>EPE / SXXK / 加工向け輸入コード
E11, E13, E15, E21, E31... これらのコードは、あなたに与えられた**「免税」という名の諸刃の剣**です。
*   **恩恵**: これらを使うことで、あなたは関税・VATの支払いを免除されます。
*   **義務**: しかしその代償として、輸入した物品のすべてが「正しく輸出向け製品に使われた」ことを、**1グラム、1ミリリットルの単位で**証明する完璧な管理体制が求められます。特に**EPE**は、その**「非関税地域」という特権**ゆえに、最も厳格な証明責任を負わされます。

<a name="audit-tips"></a>
## 5. 税務調査官を100%黙らせる、3つの鉄則

1.  **実態がコードを支配する。その逆はない**:
    契約書、インボイス、モノの流れ。その「実態」に合うコードを選ぶのです。「このコードを使いたいから」という発想は、破滅の始まりです。

2.  **現地輸出入は、宣戦布告と心得る**:
    **特にEPE**が絡む国内取引は、税関が最も注視する取引です。「国境を越える」取引と全く同じレベルの緊張感を持ってください。輸出側と輸入側の申告情報に1ミリのズレも許されません。

3.  **「なぜ、このコードなのか」を秒で答えられるように武装せよ**:
    全ての申告には、その選択を裏付ける契約書や指示書などの「証拠」を紐づけて保管してください。これが、監査の際にあなたを守る唯一の盾となります。

<a name="conclusion"></a>
## 6. 結論：あなたはリスクを放置するのか？

決定1357/QD-TCHQは、ベトナム貿易に関わるすべての経営者への**踏み絵**です。

この記事を読んで、もしあなたの会社の管理体制に少しでも不安を感じたなら、それは経営者として極めて健全な感覚です。**特にEPEの経営者であるならば、その不安は決して無視してはならない警告です。**問題は、その不安を見て見ぬふりをするかどうかです。

手遅れになってからでは、誰もあなたを救えません。

今すぐ、あなたの会社の取引が、この「法律」に100%準拠しているかを確認してください。もし1%でも疑念があるなら、即座に通関と税務の専門家に連絡してください。その決断こそが、あなたの会社を未来の破滅から救う唯一の道です。
