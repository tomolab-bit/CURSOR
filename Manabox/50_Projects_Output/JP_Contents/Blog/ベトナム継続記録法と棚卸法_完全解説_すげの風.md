こんにちは！マナラボの菅野です。

今日は、ベトナムで事業をされている方からよく質問をいただく「在庫の管理方法」についてお話しします。具体的には、「継続記録法」と「棚卸法」という2つの方法についてです。

聞き慣れない言葉かもしれませんが、簡単に言えば、在庫を「常に記録し続ける方法」か「決算時にまとめて計算する方法」かの違いです。

この記事を読むと、ベトナムで事業を営むあなたの会社がどちらの方法を使うべきか、そしてそれぞれの特徴やメリット・デメリットがわかります。ベトナム特有のルールや、業種によっては実務上どちらか一方が強く推奨される場合もあるので、今のうちに理解しておくことが大切です。

## 在庫管理には2つの方法がある

まず、在庫管理には2つの方法があるということを、ものすごく簡単に説明しますね。

### 継続記録法：常に在庫を記録し続ける方法

継続記録法というのは、商品や原材料が入ってきたとき、出て行ったときに、その都度在庫の帳簿を更新していく方法です。

例えば、あなたの会社で商品を100個仕入れたら、その瞬間に「在庫が100個増えた」と帳簿に記録します。そして、それを50個売ったら、すぐに「在庫が50個減った」と記録します。このように、取引のたびに在庫の数を更新していくのです。

だから、帳簿を見れば、今この瞬間に在庫がいくつあるかがわかります。リアルタイムで在庫が把握できるんですね。

ただし、帳簿と実際の在庫が一致しないこともあります。だから、決算のときには必ず実地棚卸といって、実際に倉庫に行って数を数えて、帳簿と照らし合わせる作業が必要になります。

この方法は、高価な商品を扱う商社や、きちんとした在庫管理が必要な製造業でよく使われています。

### 棚卸法：決算時にまとめて計算する方法

一方、棚卸法というのは、日々の在庫の増減を逐一記録しない方法です。

期間中は、仕入れた分を「仕入」という専用の勘定科目に記録していきます。そして、実際に商品が出て行ったり、使われたりしても、在庫勘定は動かしません。

決算のときになって初めて、実地棚卸を行います。倉庫に行って、実際に在庫がいくつあるかを数えます。そして、次の公式で売上原価を計算するんです。

**期首在庫 + 期中仕入 - 期末在庫 = 売上原価**

つまり、期首にあった在庫に、期中に仕入れた分を足して、期末に残っている在庫を引くと、その期間に売った商品の原価がわかるという仕組みです。

この方法は、日々の記帳が楽という利点があります。でも、期中は帳簿を見ても在庫がいくつあるかわからないというデメリットがあります。

小さな商店や、扱う商品の種類が多くて、一つ一つを逐一記録するのが大変な小売業などで使われることが多いです。

## 継続記録法と棚卸法の比較表

ここまで、2つの方法について説明してきましたが、違いがわかりにくいかもしれません。

そこで、一覧で比較できる表を作りました。あなたの会社にどちらが向いているか、この表を見ながら考えてみてください。

| 比較項目 | 継続記録法 | 棚卸法 |
|:---|:---|:---|
| **基本的な考え方** | 在庫の増減をその都度記録する | 期中は記録せず、決算時にまとめて計算する |
| **記帳のタイミング** | 仕入・出庫のたびに記録 | 仕入時のみ記録（出庫は記録しない） |
| **使用する勘定科目** | 在庫勘定（152, 156など）を直接使用 | 期中は「仕入（611）」を使用（※新通達99号では廃止） |
| **在庫の把握** | 帳簿上で常時把握可能（リアルタイム） | 期中は把握不可、決算時のみ把握可能 |
| **売上原価の計算** | 販売時に自動計算 | 決算時に「期首在庫 + 仕入 - 期末在庫」で計算 |
| **実地棚卸の頻度** | 決算時（帳簿との照合が目的） | 決算時（在庫数を確定する必要がある） |
| **日々の記帳作業** | 多い（仕入・出庫のたびに記録） | 少ない（仕入のみ記録） |
| **決算時の作業** | 実地棚卸で帳簿と照合 | 実地棚卸で在庫数を確定し、売上原価を計算 |
| **会計システムの必要性** | 高い（手作業では困難） | 低い（手作業でも可能） |
| **在庫管理の精度** | 高い（リアルタイムで把握可能） | 低い（期中は把握できない） |
| **経営判断への活用** | 高い（リアルタイムの在庫情報が活用できる） | 低い（期中は在庫情報が得られない） |
| **適用企業の傾向** | 大企業、製造業、建設業、高価値商品を扱う企業 | 小規模小売業、飲食業、多種類・低単価商品を扱う企業 |
| **適用企業の実務上の傾向** | 製造業や建設業に通常適用される（推奨） | 多種類・低単価商品を扱う小売業に通常適用される（推奨） |

この表を見ると、それぞれの特徴がはっきりしますね。

継続記録法は、リアルタイムで在庫を把握できるという大きなメリットがあります。でも、そのためには日々の記帳作業が必要になります。会計システムがあると、この作業はかなり楽になります。

一方、棚卸法は、日々の記帳が楽というメリットがあります。でも、期中は在庫がいくつあるかわからないので、経営判断に必要な情報が得られないというデメリットがあります。

どちらを選ぶかは、あなたの会社の状況によって変わってきます。

## なぜ2つの方法があるのか

実は、ベトナムの法律では、この2つの方法のどちらか一方を選んで、会計期間中は同じ方法を使い続ける必要があると決まっています。両方を混在させることはできません。

財務省通達200/2014/TT-BTCの第23条に、このことが明確に書かれています。「各企業は、継続記録法か棚卸法のいずれか一方だけを適用しなければならない」と。

ここで「会計期間中」というのは、通常は1年間（会計年度）を指しますが、会計期間が異なる場合はその期間を指します。

でも、なぜ2つの方法があるのでしょうか？

それは、企業の規模や扱う商品の性質によって、どちらの方法が向いているかが違うからです。

### 継続記録法が向いている会社

継続記録法は、以下のような会社に適しています。

- 扱う商品が高価値である
- 在庫管理を厳密に行う必要がある
- 会計システムやERPシステムが整っている
- 大企業や上場企業など、管理体制が整っている

建設業については、旧通達200号の第23条で「継続記録法は通常、製造業や建設業に適用される」と記載されています。これは法的な義務というよりも、実務上の推奨です。

なぜなら、建設業では仕掛品の管理や原価計算がとても重要だからです。決算時にまとめて計算する棚卸法では、適切な管理が難しいとされているんですね。でも、法的には建設業でも棚卸法を選ぶことは可能です。ただし、実務上は継続記録法が強く推奨されているのが現実です。

### 棚卸法が向いている会社

一方、棚卸法は、以下のような会社に適しています。

- 扱う商品の種類が非常に多く、一つ一つを記録するのが大変
- 商品の単価が低い
- 日々の記帳を簡便にしたい
- 小規模な小売業や飲食業

地方の小さな商店などでは、今でも棚卸法を使い続けているケースがあります。

## 実務ではどちらが主流なのか

ここで気になるのは、「実際にはどちらの方法を使っている会社が多いのか」ということですね。

### 伝統的な使い分け

従来は、大企業や上場企業、外資系企業など規模の大きい会社は継続記録法を、小規模な会社は棚卸法を使う傾向がありました。

小売業や飲食業では、「毎日細かく記録するより、決算時にまとめて調整した方が楽だよね」という考えから、棚卸法を使い続けてきたケースが多かったです。

### 近年の変化：継続記録法が主流に

でも、最近は状況が変わってきています。

会計ソフトやERPシステムが普及してきたことで、中小企業でも継続記録法を導入しやすくなりました。実務上、「ベトナムでは原価計算の観点から継続記録法が採用されている」という指摘もあります。

ベトナムの会計ソフト大手であるMISAも、「継続記録法による在庫管理は正確で迅速な在庫把握に寄与する有効な方法」として推奨しています。これを受けて、中小企業でも積極的に継続記録法を採用する例が増えているんです。

### 会計システムの普及による変化

なぜ、このような変化が起きているのでしょうか？

以前は、継続記録法を実践するには、手作業で一つ一つ記録していく必要がありました。それは大変な作業です。でも、今は会計ソフトがあれば、入庫や出庫のたびに自動的に在庫を更新してくれます。

システムがあることで、継続記録法の導入ハードルが下がったんです。だから、小規模な会社でも継続記録法を選びやすくなりました。

ただし、すべての会社が継続記録法に移行したわけではありません。小規模な商店や、扱う商品の種類が非常に多い小売業では、まだ棚卸法を使い続けているケースもあります。どちらの方法も、それぞれにメリット・デメリットがあるからです。

## 具体的な処理の違いを理解しよう

比較表を見ていただきましたが、実際の会計処理がどう違うのか、もう少し具体的に説明しておきますね。

### 継続記録法の場合

例えば、商品を100個仕入れた場合の仕訳はこうなります。

```
借方）商品 (TK 156)             100
借方）仮払付加価値税 (TK 133)      10
    貸方）買掛金 (TK 331)           110
```

商品を50個売った場合は、すぐに売上原価を計上します。

```
借方）売上原価 (TK 632)         50
    貸方）商品 (TK 156)           50
```

この時点で、帳簿上は在庫が50個残っていることがわかります。

### 棚卸法の場合

同じように、商品を100個仕入れた場合の仕訳はこうなります（旧通達の場合）。

```
借方）仕入 (TK 611)             100
借方）仮払付加価値税 (TK 133)      10
    貸方）買掛金 (TK 331)           110
```

商品を50個売った場合でも、在庫勘定は動かしません。売上だけを記録します。

決算時になって初めて、実地棚卸を行い、在庫が30個だったとします。すると、売上原価は次のように計算されます。

売上原価 = 期首在庫 + 期中仕入 - 期末在庫
売上原価 = 0 + 100 - 30 = 70

この計算結果で、仕入勘定から売上原価へ振り替えます。

```
借方）売上原価 (TK 632)         70
    貸方）仕入 (TK 611)           70
```

※新通達99号では、`611`勘定が廃止されたため、棚卸法でも在庫勘定を直接使用する方法に変更されています。

このように、記帳のタイミングや方法が大きく違うんですね。

## あなたの会社はどちらを選ぶべきか

さて、ここまで読んでくださったあなたは、きっと「うちの会社はどちらを選べばいいんだろう？」と思っていることでしょう。

### 選ぶときのポイント

まず、あなたの会社の状況を確認してみてください。

**継続記録法が向いているのは：**
- 扱う商品が高価値である
- 在庫管理を厳密に行う必要がある
- 会計システムやERPシステムが整っている、または導入する予定がある
- リアルタイムで在庫を把握したい
- 建設業である（実務上、継続記録法が推奨されています）

**棚卸法が向いているのは：**
- 扱う商品の種類が非常に多く、一つ一つを記録するのが大変
- 商品の単価が低い
- 日々の記帳を簡便にしたい
- 小規模な小売業や飲食業で、システム導入のコストを抑えたい

### よくある誤解

ここで、よくある誤解について整理しておきますね。

### 「継続記録法は大変そう」

確かに、継続記録法は日々の記帳が必要になります。でも、今は会計ソフトがあるので、思っているほど大変ではありません。

入庫や出庫のたびに、システムに入力するだけです。それを自動的に在庫が更新してくれます。手作業で一つ一つ記録していた時代とは、ずいぶん違います。

むしろ、リアルタイムで在庫が把握できるので、経営判断に役立つというメリットがあります。

### 「棚卸法は簡単そう」

確かに、棚卸法は日々の記帳が少ないので、一見楽に見えます。でも、決算時の実地棚卸が大変になります。

すべての在庫を正確に数えなければならないので、商品の種類が多い会社では、結構な作業量になります。また、期中は在庫がいくつあるかわからないので、経営判断に必要な情報が得られないというデメリットもあります。

### 「途中で変更できる？」

実務上、会計期間の途中で変更することは認められていないと理解されています。これは、会計の継続性の原則に基づくものです。

だから、選ぶときは慎重に考えてください。実務上、もし変更したい場合は、新しい会計年度の開始時に変更することが可能とされています。ただし、これは実務上の慣行であり、旧通達200号の第23条に明確に記載されているわけではありません。

## まとめ：あなたの会社に適した方法を選ぼう

ここまで、継続記録法と棚卸法について説明してきました。

最後に、まとめておきますね。

**継続記録法**は、在庫の増減を常に記録し続ける方法です。リアルタイムで在庫が把握でき、正確な管理ができるという利点があります。ただし、日々の記帳が必要になり、適切な会計システムや管理体制が求められます。

**棚卸法**は、決算時にまとめて計算する方法です。日々の記帳が楽ですが、期中は在庫が把握できないというデメリットがあります。決算時の実地棚卸が大変になることもあります。

ベトナムでは、この2つの方法のどちらか一方を選んで、会計期間中は同じ方法を使い続けることが法律で義務付けられています。

ただし、建設業については、旧通達200号の第23条で「継続記録法は通常、建設業に適用される」と記載されており、実務上は継続記録法が強く推奨されています。法的には建設業でも棚卸法を選ぶことは可能ですが、仕掛品の管理や原価計算の重要性から、実際には継続記録法が使われることがほとんどです。

どちらを選ぶかは、あなたの会社の規模や扱う商品の性質、管理体制などによって変わります。最近は会計システムの普及により、継続記録法を選ぶ会社が増えていますが、棚卸法が適している会社もあります。

ではまとめますね。

在庫管理は、会社の経営にとって重要な要素です。でも、それを難しく考える必要はありません。あなたの会社の状況に合わせて、適した方法を選べばいいんです。

もし、どちらを選べばいいか迷ったり、わからないことがあったら、専門家に相談してみてください。あなたの会社に最適な方法を見つけるお手伝いができるはずです。
