## 1. 通達69-2025/TT-BTCの概要と背景

### 1.1 通達の基本情報

**通達番号：** 69/2025/TT-BTC  
**発行機関：** ベトナム財務省  
**施行日：** 2025年7月1日  
**目的：** 消費税法の詳細施行に関する規定

この通達は、2025年7月1日に施行された**政令181/2025/NĐ-CP**（消費税法施行細則）の具体的な運用方法を定めた重要な文書です。

### 1.2 通達の主要な内容

通達69-2025/TT-BTCは、以下の7つの重要な内容を規定しています：

#### 📋 通達の構成

1. **非課税対象の認定手続き**（第3条：消費税法第5条の詳細規定）
2. **0%税率適用の手続き**（第4条：消費税法第9条第1項の詳細規定）
3. **%計算による課税対象**（第5条：消費税法第12条第2項b点の詳細規定）
4. **外国企業へのVAT納付代替**（第6条：消費税法第14条第2項a点の詳細規定）
5. **輸出還付の計算方法**（第7条：政令181/2025/NĐ-CP第29条第2項の詳細規定）
6. **5%税率対象の還付計算**（第8条：政令181/2025/NĐ-CP第31条の詳細規定）
7. **外国企業のVAT処理**（第9条：政令181/2025/NĐ-CP第13条、第40条第1項・第2項の詳細規定）

### 1.3 改正の背景と目的

#### 従来の課題
- 非課税対象の認定手続きが複雑
- 0%税率適用の条件が不明確
- 還付計算方法が統一されていない
- 外国企業のVAT処理が曖昧

#### 新制度の改善点
- **手続きの簡素化**：必要書類の削減
- **条件の明確化**：適用基準の統一
- **計算方法の統一**：還付額の正確な算出
- **外国企業対応**：明確な納付義務

## 2. 非課税対象の認定手続き

### 2.1 非課税対象の範囲

通達第3条では、消費税法第5条に定められた非課税対象について、以下のような詳細な認定手続きを規定しています：

#### 種畜・種苗の製品（第3条第2項a）
- **適用基準の宣言書**
- **種畜記録簿**（畜産法に基づく）
- **輸入の場合：** 原産地、品質、繁殖・育種目的の確認書（輸出国の権限機関発行）

#### 出版物の輸入（第3条第2項b）
- **新聞、雑誌、学術誌**
- **政治書、教科書、法律書**
- **科学技術書、外交用図書**
- **民族言語の図書**
- **宣伝用印刷物**
- **商業目的輸入出版物の確認書**（出版法に基づく権限機関発行）

#### 人道支援・無償援助（第3条第2項c）
- **外国組織・個人・国際機関への販売**
- **ベトナム向け人道支援・無償援助用**
- **政令181/2025/NĐ-CP第4条第11項b点に定める書類**

#### 知的財産権の移転（第3条第2項d、dd）
- **産業財産権の移転**（発明、意匠、回路配置、商標）
- **植物品種の移転**
- **保護権付与決定書**（知的財産法に基づく権限機関発行）
- **国際登録証明書**（ベトナムが締約国である国際条約）
- **移転契約書**

#### 災害・疫病・戦争からの復興支援物資（第3条第2項e）
- **自然災害、疫病、戦争からの復興支援用寄贈物資**
- **受領組織の寄贈受領承認書**

### 2.2 認定手続きの簡素化

#### 従来の手続き（複雑）
```
1. 詳細な事業計画書の提出
2. 財務諸表の3年分提出
3. 税務当局の現地調査
4. 複数部署での審査
5. 認定書の発行（3-6か月）
```

#### 新制度の手続き（簡素化）
```
1. 簡易申請書の提出
2. 基本財務資料の提出
3. 書面審査のみ
4. 認定書の発行（1-2か月）
```

### 2.3 必要書類の削減

#### 削減された書類
- **詳細事業計画書** → **事業概要書**
- **3年分財務諸表** → **直近1年分**
- **現地調査** → **書面審査のみ**
- **複数部署審査** → **一括審査**

#### 新たに必要な書類
- **事業許可証の写し**
- **税務登録証の写し**
- **非課税対象業務の説明書**

## 4. 還付計算方法の統一

### 4.1 輸出還付の計算

通達第7条およびAppendix IIでは、輸出商品・サービスの還付可能VATの計算方法を詳細に規定しています：

#### 基本計算式（Appendix II）
```
還付可能VAT = 分離可能な仕入VAT + （分離不可能な仕入VAT × 輸出売上高 ÷ 課税売上高合計）
```

#### 詳細な計算要素
1. **分離可能な仕入VAT**：輸出専用の仕入に係るVAT
2. **分離不可能な仕入VAT**：輸出・国内販売共通の仕入に係るVAT
3. **輸出売上高**：再輸出商品を除く輸出売上
4. **課税売上高合計**：課税対象商品・サービスの総売上

#### 還付条件
- **還付可能額**：300百万VND以上
- **還付上限**：輸出売上高の10%を超えない（再輸出商品除く）

### 4.2 5%税率対象の還付

通達第8条およびAppendix IIIでは、5%VAT対象商品・サービスの還付可能VATの計算方法を詳細に規定しています：

#### 基本計算式（Appendix III）
```
還付可能VAT = 分離可能な仕入VAT + （分離不可能な仕入VAT × 5%VAT対象売上高 ÷ 課税売上高合計）
```

#### 対象業務（Appendix Iより）
- **5%VAT対象商品の製造**
- **5%VAT対象サービスの提供**
- **その他の事業活動**（上記以外）

#### 還付条件
- **還付可能額**：300百万VND以上
- **計算対象**：5%VAT対象商品・サービスの製造・提供に係る仕入VAT

### 4.3 計算書類の統一

#### 必要書類
- **還付申請書**
- **売上明細書**
- **仕入明細書**
- **税務計算書**

#### 提出期限
- **月次：** 翌月15日まで
- **四半期：** 翌四半期15日まで
- **年次：** 翌年3月31日まで

## 5. 外国企業のVAT処理

### 5.1 外国企業の定義

通達第9条では、ベトナムで事業を行う外国組織・個人のVAT処理について詳細に規定しています：

#### 適用対象（第9条第1項）
a) **外国事業組織**（常設機関の有無を問わない）
b) **外国事業者**（ベトナム居住者・非居住者を問わない）
c) **ベトナムで商品流通・サービス提供の全部または一部を実施する外国組織・個人**
d) **ベトナム組織・個人を通じて契約交渉・締結を行う外国組織・個人**
e) **ベトナム市場での輸出権・輸入権・流通権を行使する外国組織・個人**

#### 適用除外（第9条第2項）
a) **ベトナム法に基づいて設立された組織**
b) **ベトナムでサービスを提供せず、外国国境検問所で商品を引き渡す場合**
c) **ベトナムでサービスを提供せず、ベトナム国境検問所で商品を引き渡す場合**
d) **海外で修理サービスを提供する場合**
e) **海外で広告・マーケティングサービスを提供する場合**（オンライン除く）
f) **海外で貿易・投資促進サービスを提供する場合**
g) **海外で仲介サービスを提供する場合**
h) **海外で研修サービスを提供する場合**（オンライン除く）
i) **海外で国際通信・郵便サービスを提供する場合**
k) **保税倉庫・内陸通関拠点を国際輸送・中継・商取引・保管・加工に使用する場合**
l) **海外で提供され、ベトナムで消費されないその他の商品・サービス**

### 5.2 VAT納付義務

#### 課税対象（第9条第3項）
a) **ベトナムで提供・使用されるサービス**（主契約・下請契約に基づく）
b) **ベトナム外で提供・ベトナムで使用されるサービス**
c) **ベトナム国内で引き渡される商品**（設置・試験・保証・保守・交換等のサービス付き）

#### VAT計算方法（第9条第4項）
a) **課税価格**：外国契約者・下請業者が受領する総収入（税額除く）
b) **VAT額**：課税価格 × VAT税率
c) **仕入VAT控除不可**：収入ベースVAT納税者は仕入VATを控除できない

#### 収入ベースVAT税率（第9条第6項）
- **一般事業**：消費税法第12条第2項b点および通達第5条に従う
- **複数事業**：各事業の課税収入に応じた税率を適用
- **建設・据付**：材料・設備込みの場合は3%VAT、材料・設備別の場合は対応税率

### 5.3 石油探査・採掘事業の特別規定

#### 石油事業のVAT処理（第9条第7項）
a) **VAT控除要件を満たさない場合**：ベトナム側が支払い前にVATを源泉徴収
b) **VAT控除要件を満たす場合**：
   - 税務登録証取得前：ベトナム側が源泉徴収
   - 税務登録証取得後：外国契約者・下請業者が自ら申告納付
   - 税務登録証取得前の仕入VATは控除不可

#### 通達第6条：外国企業へのVAT納付代替
- **ベトナム組織**が外国企業に代わってVAT納付
- **納付証明書**を取得して仕入VAT控除に使用
- **納付証明書**は国庫納付証明書として法的効力を持つ

## 7. FAQ：よくある質問と回答

### Q1. 非課税対象の認定はどのくらいの期間で完了しますか？

**A1.** **新制度では1-2か月で完了します。**

**従来：** 3-6か月  
**新制度：** 1-2か月

**理由：** 手続きの簡素化と書面審査のみへの変更

シンプルに言えば、認定期間が大幅に短縮されたということですね。

### Q2. 0%税率の適用は事前申請が必要ですか？

**A2.** **事前申請は必須ではありませんが、推奨します。**

**事前申請のメリット：**
- 税務調査での説明が容易
- 還付処理の迅速化
- リスクの最小化

**事後申請も可能：**
- 取引完了後の申請
- 適切な書類があれば承認

### Q3. 外国企業との取引でVAT納付証明書は必須ですか？

**A3.** **はい、仕入税額控除には必須です。**

**納付証明書なし：**
- 仕入税額控除不可
- VAT負担の増加
- コストアップ

**納付証明書あり：**
- 仕入税額控除可能
- VAT負担の軽減
- コスト最適化

### Q4. 還付計算の還付率はどのように決まりますか？

**A4.** **業種と業務内容により個別に算定されます。**

**算定要素：**
- **業種：** 製造業、サービス業、農業等
- **業務内容：** 具体的な事業内容
- **売上構成：** 課税・非課税の割合
- **仕入構成：** VAT課税仕入の割合

**参考還付率：**
- 製造業：5-8%
- サービス業：3-5%
- 農業：2-4%

### Q5. 新制度への移行はいつまでに完了すべきですか？

**A5.** **2025年12月末までに完了することを推奨します。**

**推奨スケジュール：**
- **2025年8月：** 現状把握と計画策定
- **2025年9-10月：** システム更新と手続き整備
- **2025年11月：** テスト運用と調整
- **2025年12月：** 本格運用開始

**理由：** 2026年からの新年度に備えた準備

### 還付計算方法

| 還付対象 | 計算式 | 還付条件 |
|----------|--------|----------|
| **輸出還付**<br>（Appendix II） | **基本計算式：**<br>還付可能VAT = 分離可能な仕入VAT + （分離不可能な仕入VAT × 輸出売上高 ÷ 課税売上高合計）<br><br>**詳細要素：**<br>・分離可能な仕入VAT：輸出専用の仕入に係るVAT<br>・分離不可能な仕入VAT：輸出・国内販売共通の仕入に係るVAT<br>・輸出売上高：再輸出商品を除く輸出売上<br>・課税売上高合計：課税対象商品・サービスの総売上 | ・還付可能額：300百万VND以上<br>・還付上限：輸出売上高の10%を超えない<br>・除外：再輸出商品の売上高 |
| **5%VAT対象還付**<br>（Appendix III） | **基本計算式：**<br>還付可能VAT = 分離可能な仕入VAT + （分離不可能な仕入VAT × 5%VAT対象売上高 ÷ 課税売上高合計）<br><br>**詳細要素：**<br>・分離可能な仕入VAT：5%VAT対象商品・サービス専用の仕入に係るVAT<br>・分離不可能な仕入VAT：5%VAT対象・その他税率対象共通の仕入に係るVAT<br>・5%VAT対象売上高：5%VAT対象商品・サービスの売上<br>・課税売上高合計：課税対象商品・サービスの総売上 | ・還付可能額：300百万VND以上<br>・計算対象：5%VAT対象商品・サービスの製造・提供に係る仕入VAT |

### 外国企業のVAT処理（第9条）

| 項目 | 内容 |
|------|------|
| **適用対象** | 外国事業組織・事業者<br>ベトナムで商品流通・サービス提供<br>ベトナム組織を通じた契約締結<br>ベトナム市場での輸出権・輸入権・流通権行使 |
| **適用除外** | ベトナム法に基づく組織<br>海外でのサービス提供<br>保税倉庫・内陸通関拠点使用<br>海外で提供・ベトナムで消費されないサービス |
| **課税対象** | ベトナムで提供・使用されるサービス<br>ベトナム外で提供・ベトナムで使用されるサービス<br>ベトナム国内引き渡し商品（サービス付き） |
| **VAT計算** | 課税価格 = 外国契約者の総収入（税額除く）<br>VAT額 = 課税価格 × VAT税率<br>仕入VAT控除不可 |

### 石油事業の特別規定（第9条第7項）

| 条件 | VAT処理方法 |
|------|-------------|
| **VAT控除要件を満たさない場合** | ベトナム側が支払い前にVAT源泉徴収 |
| **VAT控除要件を満たす場合** | 税務登録証取得前：ベトナム側が源泉徴収<br>税務登録証取得後：外国契約者が自ら申告納付<br>税務登録証取得前の仕入VATは控除不可 |
